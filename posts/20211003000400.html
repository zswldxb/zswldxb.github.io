Pytorch.torch.nn.Conv1d

Conv1d\text{Conv1d}Conv1d 网络层为一维卷积层. 输入 Input\text{Input}Input 经过网络层 Conv1d\text{Conv1d}Conv1d 后得到的输出为 Output\text{Output}Output. 一维卷积层 Conv1d\text{Conv1d}Conv1d 要求其输入 Input\text{Input}Input 为三维张量, 且其维度含义依次为小批量维度, 输入通道维度和输入特征维度, 因此 Input∈Rbatch_size×in_channels×in_features\text{Input}\in\R^{\text{batch\_size}\times\text{in\_channels}\times\text{in\_features}}Input∈Rbatch_size×in_channels×in_features, 其输出 Output\text{Output}Output 为三维张量, 其维度含义依次为小批量维度, 输出通道维度和输出特征维度, 因此 Output∈Rbatch_size×out_channels×out_features\text{Output}\in\R^{\text{batch\_size}\times\text{out\_channels}\times\text{out\_features}}Output∈Rbatch_size×out_channels×out_features.

原始的一维卷积

这里不考虑填充等其他操作, 假设一维卷积层 Conv1d\text{Conv1d}Conv1d 的卷积核 weight∈Rout_channels×in_channels×kernel_size\text{weight}\in\R^{\text{out\_channels}\times\text{in\_channels}\times\text{kernel\_size}}weight∈Rout_channels×in_channels×kernel_size, 偏置为 B∈Rout_channelsB\in\R^{\text{out\_channels}}B∈Rout_channels. Conv1d\text{Conv1d}Conv1d 网络层的定义如下:

Outputi,j,l=Bj+∑t=0in_channels−1Inputi,t,l:l+kernel_size⋆weightj,t,:\text{Output}_{i,j,l}=B_{j}+\sum_{t=0}^{\text{in\_channels}-1}\text{Input}_{i,t,l:l+\text{kernel\_size}}\star \text{weight}_{j,t,:}Outputi,j,l​=Bj​+t=0∑in_channels−1​Inputi,t,l:l+kernel_size​⋆weightj,t,:​

其中 ⋆\star⋆ 为一维互相关函数.

对于输入为 I∈Rkernel_sizeI\in\R^{\text{kernel\_size}}I∈Rkernel_size 和卷积核 K∈Rkernel_sizeK\in\R^{\text{kernel\_size}}K∈Rkernel_size, 一维卷积函数 ∘\circ∘ 定义如下:

I∘K=∑k=0kernel_size−1Ik×Kkernel_size−1−kI\circ K=\sum_{k=0}^{\text{kernel\_size}-1}I_{k}\times K_{\text{kernel\_size}-1-k}I∘K=k=0∑kernel_size−1​Ik​×Kkernel_size−1−k​

而对于输入为 I∈Rkernel_sizeI\in\R^{\text{kernel\_size}}I∈Rkernel_size 和卷积核 K∈Rkernel_sizeK\in\R^{\text{kernel\_size}}K∈Rkernel_size, 一维互相关函数 ⋆\star⋆ 定义如下:

I⋆K=∑k=0kernel_size−1Ik×KkI\star K=\sum_{k=0}^{\text{kernel\_size}-1}I_{k}\times K_{k}I⋆K=k=0∑kernel_size−1​Ik​×Kk​

由于卷积核是需要学习的参数, 对于相同的任务, 采用一维卷积函数实现后的模型的最优卷积核参数与采用互相关函数实现后的模型的最优卷积核参数仅仅是顺序反转, 其余均一样. 而且采用一维互相关函数实现的模型不用对卷积核进行反转, 实现方便. 因此卷积函数的具体实现常常采用互相关函数.

一维卷积层的相关参数

Conv1d\text{Conv1d}Conv1d 网络层的 __init__() 函数有多个参数, 不同的参数选择对应于不同的一维卷积网络层的变种.

  • in_features: 指定输入的通道数.
  • out_features: 指定输出的通道数.
  • kernel_size: 指定卷积核的大小.
  • stride: 指定步长大小.
  • padding: 指定填充大小.
  • dilation: 指定卷积核内每个元素的间隙.
  • groups: 指定输入通道到输出通道的阻塞连接数
  • bias: 指定是否使用偏置. True 为使用偏置, False 为不使用偏置.
  • padding_mode: 指定如何填充填充区的每个元素.
  • device: 指定模型所在的设备.

其中 in_features, out_features, kernel_size 在前面已经介绍过了, bias 用于指定是否使用偏置. 接下来介绍 stride, padding, dilation, groups 参数的用法.

bias 参数

bias 用于指定是否使用偏置. True 为使用偏置, False 为不使用偏置. 默认 bias=True.

stride 参数

stride 用于表示滑动的步长, 即计算一次卷积后, 输入特征图在输入特征维度方向上移动 stride 步.

padding 参数

padding 参数用于指定填充大小. 使用填充使得我们可以控制输出特征数. Pytorch 中的一维卷积层 Conv1d 拥有多种填充方式. 可以通过设置 padding='valid' 以不使用填充操作, 或使用 padding='same' 对输入特征维度进行填充使得输出特征数与输入特征数相同. 也可以设置 padding 为某一数字以在输入特征的两侧分别填充这一数字长度.

对于输入维度为 (⋅,⋅,4)(\cdot,\cdot,4)(⋅,⋅,4), 卷积核维度为 (⋅,⋅,3)(\cdot,\cdot,3)(⋅,⋅,3), 步长为 111, 如果设置 padding='valid', 则不进行填充, 则输出维度为 (⋅,⋅,2)(\cdot,\cdot,2)(⋅,⋅,2), 其中输出特征数为 ⌊4−(3−1)−11+1⌋=2\lfloor\dfrac{4-(3-1)-1}{1}+1\rfloor=2⌊14−(3−1)−1​+1⌋=2.

对于输入维度为 (⋅,⋅,4)(\cdot,\cdot,4)(⋅,⋅,4), 卷积核维度为 (⋅,⋅,3)(\cdot,\cdot,3)(⋅,⋅,3), 步长为 111, 如果设置 padding='same', 则进行填充, 其输出维度为 (⋅,⋅,4)(\cdot,\cdot,4)(⋅,⋅,4).

对于输入维度为 (⋅,⋅,4)(\cdot,\cdot,4)(⋅,⋅,4), 卷积核维度为 (⋅,⋅,3)(\cdot,\cdot,3)(⋅,⋅,3), 步长为 111, 如果设置 padding=1, 则将输入特征的左右两侧分别填充长度为 111. 使其维度变成 (⋅,⋅,6)(\cdot,\cdot,6)(⋅,⋅,6), 其中输出特征数为 ⌊6−(3−1)−11+1⌋=4\lfloor\dfrac{6-(3-1)-1}{1}+1\rfloor=4⌊16−(3−1)−1​+1⌋=4.

注意 padding 参数用于确定需要填充的长度, 而具体填充什么值则是通过 padding_mode 来确定.

dilation 参数用于指定计算时卷积核内每个元素之间的距离. 默认为 111. 具体过程见演示.

groups 参数

此前的 stride, padding, dilation 参数作用于特征维度, 而不改变通道维度的计算方式, 而 groups 参数作用于通道维度, 而不改变特征维度的计算.

Pytorch 官方实现关于 groups 参数的介绍如下

例如当 groups=4 时, 对于输入维度为 (⋅,12,⋅)(\cdot,12,\cdot)(⋅,12,⋅), 输出维度为 (⋅,24,⋅)(\cdot,24,\cdot)(⋅,24,⋅). 此时按照通道维度将输入分为四组, 输出分为四组. 输入通道的四组的下标分别为 0-2,3-5,6-8,9-11, 输出通道的四组的下标分别为 0-5,6-11,12-17,18-23. 然后四组输入数据分别输入到四个不同的卷积核, 其输出通道数为 666, 然后得到四组输出, 然后将其按通道维度进行拼接.

值得注意的是, 设置 groups 时需要满足 in_features 和 out_features 都是 groups 的倍数.

当 in_features==groups 且 out_features==K*groups 时, 称其为深度可分离卷积.
