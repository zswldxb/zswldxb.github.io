P1025 [NOIP2001 提高组] 数的划分

题目描述

将整数 nnn 分成 kkk 份，且每份不能为空，任意两个方案不相同(不考虑顺序)。

例如：n=7,k=3n=7,k=3n=7,k=3，下面三种分法被认为是相同的。

1,1,51,5,15,1,1\begin{aligned} &1,1,5\\ &1,5,1\\ &5,1,1 \end{aligned}​1,1,51,5,15,1,1​

问有多少种不同的分法。

输入格式

n,k(6<n≤200,2≤k≤6)n,k(6<n \leq 200,2 \leq k \leq 6)n,k(6<n≤200,2≤k≤6)。

输出格式

111 个整数，即不同的分法。

输入输出样例

输入 #1\#1#1

输出 #1\#1#1

说明/提示

四种分法为：

1,1,51,2,41,3,32,2,3\begin{aligned} &1,1,5\\ &1,2,4\\ &1,3,3\\ &2,2,3\\ \end{aligned}​1,1,51,2,41,3,32,2,3​

思路

思路一

记 f(n,k)f(n,k)f(n,k) 表示将整数 nnn 分成 kkk 份，且每份不能为空，任意两个方案不相同(不考虑顺序)的分法。

当 n=7,k=3n=7,k=3n=7,k=3 时，一共有四种分法，分别为：{1,1,5}\{1,1,5\}{1,1,5}，{1,2,4}\{1,2,4\}{1,2,4}，{1,3,3}\{1,3,3\}{1,3,3}，{2,2,3}\{2,2,3\}{2,2,3}。

我们可以把这四种分法分为两组，其中第一组（记为 AAA）中的每个分法都含有 111，第二组（记为 BBB）中的每个分法都不包括 111。所以

A={{1,1,5},{1,2,4},{1,3,3}}B={{2,2,3}}\begin{aligned} &A =\{ \{1,1,5\},\{1,2,4\},\{1,3,3\}\}\\ &B =\{\{2,2,3\}\} \end{aligned}​A={{1,1,5},{1,2,4},{1,3,3}}B={{2,2,3}}​

因为 AAA 组中每一个分法都包含 111 所以 AAA 组分法的个数等于将每个分组中去除一个 111 后的分法的个数，即 {{1,5},{2,4},{3,3}}\{ \{1,5\},\{2,4\},\{3,3\}\}{{1,5},{2,4},{3,3}} 的个数，而这等于 n=6,k=2n=6,k=2n=6,k=2 时的分法个数。

因为 BBB 组中每一个分法都不包含 111，为了使分法包含 111，我们考虑将每一个分法的每一个元素都减少 111，这样处理后的分法就可能会包含 111， 处理后的分法为 {{1,1,2}}\{\{1,1,2\}\}{{1,1,2}}，这个分法的个数等同于 n=4,k=3n=4,k=3n=4,k=3 时的分法个数。所以 f(7,3)=f(6,2)+f(4,3)f(7,3) = f(6,2)+f(4,3)f(7,3)=f(6,2)+f(4,3)。

同理我们可以得到 f(n,k)=f(n−1,k−1)+f(n−k,k)f(n,k) = f(n-1,k-1)+f(n-k,k)f(n,k)=f(n−1,k−1)+f(n−k,k)。

接下来我们考虑边界情况。

当 n<kn<kn<k 时，由于每份不能为空，所以此时 f(n,k)=0f(n,k)=0f(n,k)=0。

当 n=kn=kn=k 时，由于每份不能为空，所以此时 f(n,k)=1f(n,k)=1f(n,k)=1。

当 k=1k=1k=1 时，由于只有一份，所以 f(n,k)=1f(n,k)=1f(n,k)=1。

综上我们有

f(n,k)={1,k=1,n≥11,k=n,n>1f(n−1,k−1)+f(n−k,k),1<k<n,n>2f(n,k) = \begin{cases} &1,&k=1,n\geq1\\ &1,&k=n,n>1\\ &f(n-1,k-1)+f(n-k,k),&1<k<n,n>2\\ \end{cases}f(n,k)=⎩⎪⎨⎪⎧​​1,1,f(n−1,k−1)+f(n−k,k),​k=1,n≥1k=n,n>11<k<n,n>2​

思路二

对于给定的 nnn 和 kkk，根据要求我们可以设一个分法为 x1,x2,⋯ ,xkx_{1},x_{2},\cdots,x_{k}x1​,x2​,⋯,xk​，其中 1≤xi≤xj,(1≤i<j≤k)1\leq x_{i}\leq x_{j},(1\leq i < j\leq k)1≤xi​≤xj​,(1≤i<j≤k)，且 ∑i=1kxi=n\displaystyle{\sum_{i=1}^{k} x_{i} = n}i=1∑k​xi​=n。设解的个数为 f(n,k)f(n,k)f(n,k)。分两种情况。

情况一：x1=1x_{1} = 1x1​=1，此时 ∑i=1kxi=x1+∑i=2kxi=1+∑i=2kxi=n\displaystyle{\sum_{i=1}^{k} x_{i} = x_{1}+\sum_{i=2}^{k}x_{i}= 1+\sum_{i=2}^{k}x_{i} = n}i=1∑k​xi​=x1​+i=2∑k​xi​=1+i=2∑k​xi​=n，即 ∑i=2kxi=n−1\displaystyle{\sum_{i=2}^{k}x_{i}=n-1}i=2∑k​xi​=n−1，其中 1≤x1=1≤xj,(1≤1<j≤k)1\leq x_{1}=1\leq x_{j},(1\leq 1<j\leq k)1≤x1​=1≤xj​,(1≤1<j≤k) 且 1≤xi≤xj,(2≤i<j≤k)1\leq x_{i}\leq x_{j},(2\leq i < j\leq k)1≤xi​≤xj​,(2≤i<j≤k)，即 1≤xi≤xj,(2≤i<j≤k)1\leq x_{i}\leq x_{j},(2\leq i < j\leq k)1≤xi​≤xj​,(2≤i<j≤k)。这时我们设置一个映射 xi→yi−1,(2≤i≤k)x_{i}\rightarrow y_{i-1},(2\leq i\leq k)xi​→yi−1​,(2≤i≤k)，则可将情况一描述为一个分法 y1,y2,⋯yk−1y_{1},y_{2},\cdots y_{k-1}y1​,y2​,⋯yk−1​，其中 1≤yi<yj,(1≤i<j≤k)1\leq y_{i}<y_{j},(1\leq i< j \leq k)1≤yi​<yj​,(1≤i<j≤k)，且 ∑i=1k−1yi=n−1\displaystyle{\sum_{i=1}^{k-1}y_{i}=n-1}i=1∑k−1​yi​=n−1。所以该情况的分组个数为 f(n−1,k−1)f(n-1,k-1)f(n−1,k−1)。

情况二：x1>1x_{1}>1x1​>1，这时 1<x1≤xj,(1<j≤k)1< x_{1} \leq x_{j},(1<j\leq k)1<x1​≤xj​,(1<j≤k)，即 2≤xi≤xj,(1≤i<j≤k)2\leq x_{i}\leq x_{j},(1\leq i<j\leq k)2≤xi​≤xj​,(1≤i<j≤k)。由于分法中的每一个元素都大于 111，我们设置一个映射 xi→zi+1,(1≤i≤k)x_{i}\rightarrow z_{i}+1,(1\leq i \leq k)xi​→zi​+1,(1≤i≤k)，此时 ∑i=1kxi=∑i=1k(zi+1)=k+∑i=1kzi=n\displaystyle{\sum_{i=1}^{k}x_{i} = \sum_{i=1}^{k}(z_{i}+1)=k+\sum_{i=1}^{k}z_{i}=n}i=1∑k​xi​=i=1∑k​(zi​+1)=k+i=1∑k​zi​=n，即 ∑i=1kzi=n−k\displaystyle{\sum_{i=1}^{k}z_{i}=n-k}i=1∑k​zi​=n−k，其中 1≤zi≤zj,(1≤i<j≤k)1\leq z_{i}\leq z_{j},(1\leq i<j\leq k)1≤zi​≤zj​,(1≤i<j≤k)。所以该情况的分组个数为 f(n−k,k)f(n-k,k)f(n−k,k)。

所以 f(n,k)=f(n−1,k−1)+f(n−k,k)f(n,k) = f(n-1,k-1)+f(n-k,k)f(n,k)=f(n−1,k−1)+f(n−k,k)。边界情况同思路一。

综上
