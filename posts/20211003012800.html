Pytorch.torch.nn.PairwiseDistance

网络层 PairwiseDistance\text{PairwiseDistance}PairwiseDistance 将计算两个张量之间的距离.

p 参数

p 参数用于指定使用 p 范数.

eps 参数

eps 参数用于避免分母为 000.

keepdim 参数

keepdim=True 时, 输出与输入张量的维度数相同(形状不同), keepdim=False 时, 输出与输入张量的维度数不同.

对于 keepdim=False 时, 输入为 X,Z∈Rd1×⋯×dnX,Z\in\R^{d_{1}\times\cdots\times d_{n}}X,Z∈Rd1​×⋯×dn​, 输出为 Y∈Rd1×⋯×dn−1Y\in\R^{d_{1}\times\cdots\times d_{n-1}}Y∈Rd1​×⋯×dn−1​, 有

Y=\begin{bmatrix}\sqrt[p]{\displaystyle\sum_{i_{n}=0}^{d_{n}-1}|X_{i_{1},\cdots,i_{n-1},i_{n}}-Z_{i_{1},\cdots,i_{n-1},i_{n}}|^{p}}\end{bmatrix}_{\begin{subarray}{c}0\leq i_{1}<d_{1}\\\vdots\\0\leq i_{n-1}<d_{n-1}\end{subarray}}

示例

import torch
import torch.nn as nn

print(f'torch version :{torch.__version__}')


def pairwise_dis(x1, x2, p, eps, keepdim):
    y_hat = torch.sum(torch.abs(x1 - x2 + eps) ** p , dim=-1, keepdim=keepdim) ** (1 / p)
    return y_hat

def solve(features, p=2, eps=1e-6, keepdim=False):
    input1 = torch.rand(features)
    input2 = torch.rand(features)
    layer = nn.PairwiseDistance(p=p, eps=eps, keepdim=keepdim)

    y = layer(input1, input2)

    y_hat = pairwise_dis(input1, input2, p, eps, keepdim)

    print(f'error: {torch.abs(y-y_hat).flatten()}')


if __name__ == '__main__':
    features = (5, 8, 4)

    solve(features, p=2, eps=1e-6, keepdim=False)

    solve(features, p=1, eps=1e-6, keepdim=False)

    solve(features, p=2, eps=1e-6, keepdim=True)
