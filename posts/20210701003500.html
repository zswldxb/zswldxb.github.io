Self-Attention with Relative Position Representations

Preface

Paper

Motivation

在计算 Multi-Head Attention 时, 原始的 Transformer 没有考虑序列中的两个 Token 的相对位置关系, 本文将相对位置编码引入 Multi-Head Attention 中.

Background

对于输入序列 X=(x1,x2,⋯ ,xn)X=(x_{1},x_{2},\cdots,x_{n})X=(x1​,x2​,⋯,xn​), 其中 xi∈Rdxx_{i}\in\R^{d_{x}}xi​∈Rdx​, 其经过 Multi-Head Attention 后得到 Z=(z1,z2,⋯ ,zn)Z=(z_{1},z_{2},\cdots,z_{n})Z=(z1​,z2​,⋯,zn​), 其中 zi∈Rdzz_{i}\in\R^{d_{z}}zi​∈Rdz​.

对于每个输出向量 ziz_{i}zi​, 其可通过对输入线性变换后的结果进行加权和得到:

zi=∑j=1nαij(xjWV)z_{i}=\sum_{j=1}^{n}\alpha_{ij}(x_{j}W^{V})zi​=j=1∑n​αij​(xj​WV)

而每个加权系数 αij\alpha_{ij}αij​ 可以通过 softmax\text{softmax}softmax 得到:

αij=exp⁡(eij)∑k=1nexp⁡(eik)\alpha_{ij}=\dfrac{\exp{(e_{ij})}}{\displaystyle{\sum_{k=1}^{n}\exp{(e_{ik})}}}αij​=k=1∑n​exp(eik​)exp(eij​)​

而每个 eije_{ij}eij​ 可由 QQQ 与 KKK 的 compatibility function 计算得到, 在 Transformer 中选择 Scaled dot product 作为 compatibility function, 因此 eije_{ij}eij​ 可由如下公式计算得到(线性变换是为了增加输入的表示能力):

eij=(xiWQ)(xjWK)Tdze_{ij}=\frac{(x_{i}W^{Q})(x_{j}W^{K})^{T}}{\sqrt{d_{z}}}eij​=dz​​(xi​WQ)(xj​WK)T​

其中 WQ,WK,WV∈Rdx×dzW^{Q},W^{K},W^{V}\in\R^{d_{x}\times d_{z}}WQ,WK,WV∈Rdx​×dz​ 是可学习的参数矩阵, 这些参数矩阵对于每个层和每个注意力头中是不一样的.

Method

本文注意到在计算 Multi-Head Attention 时, 没有考虑序列中 Token 之间的位置关系.

本文设置两个表示第 iii 个输入 Token 与第 jjj 个 Token 之间的位置向量 aijV,aijK∈Rdaa_{ij}^{V},a_{ij}^{K}\in\R^{d_{a}}aijV​,aijK​∈Rda​. 其中 da=dzd_{a}=d_{z}da​=dz​

本分分别将 aijVa_{ij}^{V}aijV​ 和 aijKa_{ij}^{K}aijK​ 加入到注意力计算公式中:

zi=∑j=1nαij(xjWV+aijV)z_{i}=\sum_{j=1}^{n}\alpha_{ij}(x_{j}W^{V}+a_{ij}^{V})zi​=j=1∑n​αij​(xj​WV+aijV​)

和

eij=(xiWQ)(xjWK+aijK)Tdze_{ij}=\frac{(x_{i}W^{Q})(x_{j}W^{K}+a_{ij}^{K})^{T}}{\sqrt{d_{z}}}eij​=dz​​(xi​WQ)(xj​WK+aijK​)T​

本文使用相对位置编码. 设置最大的相对距离为 kkk.

对于任意的第 iii 个输入 Token 与第 jjj 个 Token, 其 aijVa_{ij}^{V}aijV​ 和 aijWa_{ij}^{W}aijW​ 可由如下计算公式得到:

aijV=wclip(j−i,k)WaijK=wclip(j−i,k)Vclip(x,k)=max⁡(−k,min⁡(k,x))\begin{aligned} a_{ij}^{V}=&w_{\text{clip}(j-i, k)}^{W}\\ a_{ij}^{K}=&w_{\text{clip}(j-i, k)}^{V}\\ \text{clip}(x,k)=&\max(-k,\min(k, x))\\ \end{aligned}aijV​=aijK​=clip(x,k)=​wclip(j−i,k)W​wclip(j−i,k)V​max(−k,min(k,x))​

这样相同的距离之差的任意两个 Token 组被分配到了同一组, 不同的距离之差的任意两个 Token 组被分配到不同组.

这里对距离之差的绝对值超出 kkk 的所有相对距离进行了截断.

因此可学习参数变为 wK=(w−kK,⋯ ,wkK)w^{K}=(w_{-k}^{K},\cdots,w_{k}^{K})wK=(w−kK​,⋯,wkK​) 和 wV=(w−kV,⋯ ,wkV)∈R(2k+1)×daw^{V}=(w_{-k}^{V},\cdots,w_{k}^{V})\in\R^{(2k+1)\times d_{a}}wV=(w−kV​,⋯,wkV​)∈R(2k+1)×da​.

本文的相对位置编码在不同的注意力头之间共享.
