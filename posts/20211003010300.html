Pytorch.torch.nn.Upsample

网络层 Upsample\text{Upsample}Upsample 为上采样层. 输入 Input\text{Input}Input 经过网络层 Upsample\text{Upsample}Upsample 后得到的输出为 Output\text{Output}Output. 网络层 Upsample\text{Upsample}Upsample 要求其输入 Input\text{Input}Input 为三维(时序数据), 四维(图像数据)或五维(空间数据)张量.

其重要参数如下:

  • size: 表示输出的特征维度大小(不包括小批量数和通道数).
  • scale_factor: 表示相对于输入特征维度大小的放缩比例.
  • mode: 表示使用什么插值算法. 可使用的插值算法有 nearest, linear, bilinear, bicubic 和 trilinear. 默认为 nearest.
  • align_corners: 表示是否对齐输入与输出的每个角. 默认是 False.

示例

import torch
import torch.nn as nn

print(f'torch version: {torch.__version__}')
# torch version: 1.10.2

batch_size = 1
in_channels = 2
in_height = 2
in_width = 2

input = torch.arange(batch_size*in_channels*in_height*in_width, dtype=torch.float32).reshape((batch_size, in_channels, in_height, in_width))
print(f'input: {input}')
'''
input: tensor([[[[0., 1.],
          [2., 3.]],

         [[4., 5.],
          [6., 7.]]]])
'''

layer = nn.Upsample(scale_factor=2)
y = layer(input)
print(f'output: {y}')
'''
output: tensor([[[[0., 0., 1., 1.],
          [0., 0., 1., 1.],
          [2., 2., 3., 3.],
          [2., 2., 3., 3.]],

         [[4., 4., 5., 5.],
          [4., 4., 5., 5.],
          [6., 6., 7., 7.],
          [6., 6., 7., 7.]]]])
'''

layer = nn.Upsample(scale_factor=2, mode='bilinear')
y = layer(input)
print(f'output: {y}')
'''
output: tensor([[[[0.0000, 0.2500, 0.7500, 1.0000],
          [0.5000, 0.7500, 1.2500, 1.5000],
          [1.5000, 1.7500, 2.2500, 2.5000],
          [2.0000, 2.2500, 2.7500, 3.0000]],

         [[4.0000, 4.2500, 4.7500, 5.0000],
          [4.5000, 4.7500, 5.2500, 5.5000],
          [5.5000, 5.7500, 6.2500, 6.5000],
          [6.0000, 6.2500, 6.7500, 7.0000]]]])
'''
