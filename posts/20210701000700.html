You Only Look Once: Unified, Real-Time Object Detection

Preface

Paper

Purpose

本文指出: 本文思路与此前的目标检测的思路不同, 本文思路将目标检测当作回归问题处理, 直接根据网络预测边界框和类概率. 本文提出的网络名取为 YOLO.

YOLO 相比于其他目标检测方法的优点:

  1. YOLO 的检测速度很快.
  2. YOLO 在进行预测时是基于整个图像的.
  3. YOLO 学习到了目标的通用表示.
  4. YOLO 是 end-to-end model.

YOLO 相比于其他目标检测方法的缺点:

  1. 对边界框预测强加了空间约束, 因为每个网格单元只能预测两个框, 并且只能有一类. 这大大限制了模型可以预测的目标数量.
  2. 模型定位不准确.

Methodology

本文首先规定: 将输入图像划分为 S×SS\times SS×S 的网格. 若对象的中心落入网格单元, 则该网格单元负责检测该对象.

每个网格都负责预测 BBB 个 bounding box 和这些 box 的置信度(confidence). 本文定义置信度为

Pr(Object)×IOUpredtruthPr(Object)\times IOU_{pred}^{truth}Pr(Object)×IOUpredtruth​

若网格中不存在对象, 则置信度为零, 否则, 置信度为预测 box 与 ground truth 的 IOU. 因此, 每个 bounding box 由五个部分组成, x,y,w,hx,y,w,hx,y,w,h 和 confidence. 其中 (x,y)(x,y)(x,y) 表示 box 相对于网格中心的偏移, www 和 hhh 表示相对于整个图像的高度和宽度比例. confidence 表示预测 box 与 ground truth 的 IOU.

此外, 每个网格还负责预测 CCC 个 conditional class probabilities Pr(Classi∣Object)Pr(Class_{i}|Object)Pr(Classi​∣Object). 此时假定网格中包含了一个目标. 此时的条件类别概率与 BBB 无关.

在测试时, 特定类别的置信度定义为如下:

Pr(Classi)×IOUpredtruth=Pr(Classi∣Object)×Pr(Object)×IOUpredtruthPr(Class_{i})\times IOU_{pred}^{truth}=Pr(Class_{i}|Object)\times Pr(Object)\times IOU_{pred}^{truth}Pr(Classi​)×IOUpredtruth​=Pr(Classi​∣Object)×Pr(Object)×IOUpredtruth​

这里假定测试样本的分布与训练样本分布相同.

loss function

本文损失函数使用 sum-squared error, 其易于优化. 本文通过修改损失函数以解决引入 sum-squared error 而产生的问题.

  1. 本文增加 box 坐标预测的损失, 并减少不包含目标的网格的置信度损失. 分别设为 λcoord=5\lambda_{coord}=5λcoord​=5 和 λnoobj=0.5\lambda_{noobj}=0.5λnoobj​=0.5. 以解决定位误差和可能不理想的分类误差的等权相加, 以及某些图像的某些网格单元不包含对象而导致这些网格的置信度趋于零.
  2. 本文通过预测 bounding box 的宽度和高度的平方根. 来解决大 box 和 小 box的重要程度相同的问题, 经过这样处理后, 大 box 比小 box 更重要.
