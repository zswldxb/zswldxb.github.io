Swin Transformer: Hierarchical Vision Transformer using Shifted Windows

Preface

Paper

Motivation

将应用于 NLP 中的 Transformer 应用到 CV 中, 会存在诸多的问题, 如此前提出的 Vision Transformer, Vision Transformer 将一张图片分成 NNN 个 Patch, 将每个 Patch 视为一个 token, 然后引入多头注意力机制来提取 token 之间的关联信息. 然而由于 NNN 通常很大, 而在计算注意力时, 其计算复杂度为 O(N2)O(N^{2})O(N2) 时间开销过大.

本文提出的 Swin Transformer 主要是通过将卷积神经网络的思想与 Transformer 结合起来, 将整个模型分为不同的 stage, 每个 stage 内只计算大小为 (window×window)(\text{window}\times\text{window})(window×window) 的 block 之间的注意力, 且在每个 stage结束前使用下采样(最后一个除外). 此外为了模拟堆叠 Conv 以增大感受野的方法, 本文还提出了 Shifted Window based Self-Attention.

本文的模型主体部分实现了用 TransformerBlock 替换 ConvBlock 的方法.
