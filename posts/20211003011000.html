Pytorch.torch.nn.PoissonNLLLoss

损失函数层 PoissonNLLLoss\text{PoissonNLLLoss}PoissonNLLLoss 为真实标签服从泊松分布的负对数似然损失函数.

对于随机变量 yyy, 假设其服从泊松分布, 则

P(y=k;λ)=λkk!exp⁡(−λ)P(y=k;\lambda)=\dfrac{\lambda^{k}}{k!}\exp{(-\lambda)}P(y=k;λ)=k!λk​exp(−λ)

损失函数层 PoissonNLLLoss\text{PoissonNLLLoss}PoissonNLLLoss 其要求输入张量 XXX 与标签张量 YYY 均为 nnn 维张量, 且 X,Y∈Rd1×⋯×dnX,Y\in\R^{d_{1}\times\cdots\times d_{n}}X,Y∈Rd1​×⋯×dn​.

reduction: 用于指定计算的模式, 'none', 'mean' 或 'sum'. 默认为 'mean'.

log_input 参数

当 log_input=True 时, 将使用 exp⁡(X)\exp(X)exp(X) 代替 XXX, 如当 reduction='none', log_input=True 时. 其计算过程如下:

\begin{aligned} \text{PoissonNLLLoss}(X,Y)=&\begin{bmatrix}\exp{(X_{i_{1},\cdots,i_{n}})}-Y_{i_{1},\cdots,i_{n}}\log{(\exp{(X_{i_{1},\cdots,i_{n}})})}+\log{(Y_{i_{1},\cdots,i_{n}}!)}\end{bmatrix}_{\begin{subarray}{c}0\leq i_{1}<d_{1}\\\vdots\\0\leq i_{n}<d_{n}\end{subarray}}\\ =&\begin{bmatrix}\exp{(X_{i_{1},\cdots,i_{n}})}-Y_{i_{1},\cdots,i_{n}}X_{i_{1},\cdots,i_{n}}+\log{(Y_{i_{1},\cdots,i_{n}}!)}\end{bmatrix}_{\begin{subarray}{c}0\leq i_{1}<d_{1}\\\vdots\\0\leq i_{n}<d_{n}\end{subarray}}\\ \end{aligned}

当 log_input=False 时, 将还是使用 XXX 直接进行计算.

eps 参数用于防止当 Xi1,⋯ ,in=0X_{i_{1},\cdots,i_{n}}=0Xi1​,⋯,in​​=0 时, log⁡(Xi1,⋯ ,in)\log{(X_{i_{1},\cdots,i_{n}})}log(Xi1​,⋯,in​​) 出错, 因此在计算时使用 log⁡(Xi1,⋯ ,in+eps)\log{(X_{i_{1},\cdots,i_{n}}+\text{eps})}log(Xi1​,⋯,in​​+eps) 代替.

示例

import torch
import torch.nn as nn

print(f'torch version: {torch.__version__}')
# torch version: 1.10.2

# torch.set_printoptions(sci_mode=False)


def poisson_nll(input, target, reduction, log_input, full, eps):
    if log_input:
        input = input.exp()
    y_hat = input - target * torch.log(input + eps)
    if full:
        tmp = target * torch.log(target) - target + 0.5 * torch.log(2 * torch.pi * target)
        y_hat += torch.where(target > 1., tmp, torch.zeros_like(tmp))

    if reduction == 'mean':
        y_hat = torch.mean(y_hat)
    elif reduction == 'sum':
        y_hat = torch.sum(y_hat)

    return y_hat

def solve( features, reduction='mean', log_input=False, full=False, eps = 1e-8):
    reduction_list = ['none', 'mean', 'sum']
    assert reduction in reduction_list, 'mode error'

    input = torch.rand(features)
    target = (torch.rand(features) * 5).long().to(torch.float32)

    loss = nn.PoissonNLLLoss(reduction=reduction, log_input=log_input, full=full, eps=eps)
    y = loss(input, target)

    y_hat = poisson_nll(input, target, reduction, log_input, full, eps)

    print(f'error: {(torch.abs(y-y_hat)).flatten()}')


if __name__ == '__main__':
    features = (10, 10)
    
    solve(features, reduction='none', log_input=False, full=False, eps = 1e-8)
    
    solve(features, reduction='mean', log_input=False, full=False, eps = 1e-8)
    
    solve(features, reduction='sum', log_input=False, full=False, eps = 1e-8)
    
    solve(features, reduction='none', log_input=True, full=False, eps = 1e-8)
    
    solve(features, reduction='mean', log_input=True, full=False, eps = 1e-8)
    
    solve(features, reduction='sum', log_input=True, full=False, eps = 1e-8)
    
    solve(features, reduction='none', log_input=True, full=True, eps = 1e-8)
    
    solve(features, reduction='mean', log_input=True, full=True, eps = 1e-8)
    
    solve(features, reduction='sum', log_input=True, full=True, eps = 1e-8)
