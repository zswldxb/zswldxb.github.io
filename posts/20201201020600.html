2.6 Special Kinds of Matrices and Vectors(特殊类型的矩阵和向量)

Some special kinds of matrices and vectors are particularly useful.

有些特殊类型的矩阵和向量是特别有用的。

Diagonal matrices consist mostly of zeros and have nonzero entries only along the main diagonal. Formally, a matrix DDD is diagonal if and only if Di,j=0D_{i,j}= 0Di,j​=0 for all i≠ji\neq ji​=j. We have already seen one example of a diagonal matrix: the identity matrix, where all the diagonal entries are 1. We write diag(v)\mathrm{diag}(v)diag(v) to denote a square diagonal matrix whose diagonal entries are given by the entries of the vector vvv . Diagonal matrices are of interest in part because multiplying by a diagonal matrix is computationally eﬃcient. To compute diag(v)x\mathrm{diag}(v)xdiag(v)x, we only need to scale each element xix_{i}xi​ by viv_{i}vi​. In other words, diag(v)x=v⊙x\mathrm{diag}(v)x=v\odot xdiag(v)x=v⊙x. Inverting a square diagonal matrix is also eﬃcient. The inverse exists only if every diagonal entry is nonzero, and in that case, diag(v)−1=diag([1/v1,⋯ ,1/vn]T)\mathrm{diag}(v)^{−1}=\mathrm{diag}([1/v_{1},\cdots,1/v_{n}]^{T})diag(v)−1=diag([1/v1​,⋯,1/vn​]T). In many cases, we may derive some general machine learning algorithm in terms of arbitrary matrices but obtain a less expensive (and less descriptive) algorithm by restricting some matrices to be diagonal.

对角矩阵（diagonal matrix）只在主对角线上含有非零元素，其他位置都是零。形式上，矩阵 DDD 是对角矩阵，当且仅当对于所有的 i≠j,Di,j=0i\neq j,D_{i,j}= 0i​=j,Di,j​=0。我们已经看到过一个对角矩阵：单位矩阵，其对角元素全部是 111。我们用 diag(v)\mathrm{diag}(v)diag(v) 表示对角元素由向量 vvv 中元素给定的一个对角方阵。对角矩阵受到关注的部分原因是对角矩阵的乘法计算很高效。计算乘法 diag(v)x\mathrm{diag}(v)xdiag(v)x ，我们只需要将 xxx 中的每个元素 xix_{i}xi​ 放大 viv_{i}vi​ 倍。换言之，diag(v)x=v⊙x\mathrm{diag}(v)x=v\odot xdiag(v)x=v⊙x。计算对角方阵的逆矩阵也很高效。对角方阵的逆矩阵存在，当且仅当对角元素都是非零值，在这种情况下， diag(v)−1=diag([1/v1,⋯ ,1/vn]T)\mathrm{diag}(v)^{−1}=\mathrm{diag}([1/v_{1},\cdots,1/v_{n}]^{T})diag(v)−1=diag([1/v1​,⋯,1/vn​]T)。在很多情况下，我们可以根据任意矩阵导出一些通用的机器学习算法，但通过将一些矩阵限制为对角矩阵，我们可以得到计算代价较低的（并且简明扼要的）算法。

Not all diagonal matrices need be square. It is possible to construct a rectangular diagonal matrix. Nonsquare diagonal matrices do not have inverses, but we can still multiply by them cheaply. For a nonsquare diagonal matrix DDD, the product DxDxDx will involve scaling each element of xxx and either concatenating some zeros to the result, if DDD is taller than it is wide, or discarding some of the last elements of the vector, if D is wider than it is tall.

并非所有的对角矩阵都是方阵。长方形的矩阵也有可能是对角矩阵。非方阵的对角矩阵没有逆矩阵，但我们仍然可以高效地计算它们的乘法。对于一个长方形对角矩阵 DDD 而言，乘法 DxDxDx 会涉及 xxx 中每个元素的缩放，如果 DDD 是瘦长型矩阵，那么在缩放后的末尾添加一些零；如果 DDD 是胖宽型矩阵，那么在缩放后去掉最后一些元素。

A symmetric matrix is any matrix that is equal to its own transpose:

对称（symmetric）矩阵是转置和自己相等的矩阵，即

A=AT.(2.35)A=A^{T}.\qquad\qquad\qquad(2.35)A=AT.(2.35)

Symmetric matrices often arise when the entries are generated by some function of two arguments that does not depend on the order of the arguments. For example, if AAA is a matrix of distance measurements, with Ai,jA_{i,j}Ai,j​ giving the distance from point iii to point jjj, then Ai,j=Aj,iA_{i,j}= A_{j,i}Ai,j​=Aj,i​ because distance functions are symmetric.

当某些不依赖参数顺序的双参数函数生成元素时，对称矩阵经常会出现。例如，如果 AAA 是一个距离度量矩阵，Ai,jA_{i,j}Ai,j​ 表示点i到点j的距离，那么 Ai,j=Aj,iA_{i,j}= A_{j,i}Ai,j​=Aj,i​ ，因为距离函数是对称的。

A unit vector is a vector with unit norm:

单位向量（unit vector）是具有单位范数（unit norm）的向量，即

∥x∥2=1.(2.36)\parallel x\parallel_{2}=1.\qquad\qquad\qquad(2.36)∥x∥2​=1.(2.36)

A vector xxx and a vector yyy are orthogonal to each other if xTy=0x^{T}y=0xTy=0. If both vectors have nonzero norm, this means that they are at a 90 degree angle to each other. In Rn\R^{n}Rn, at most nnn vectors may be mutually orthogonal with nonzero norm. If the vectors not only are orthogonal but also have unit norm, we call them orthonormal.

如果 xTy=0x^{T}y=0xTy=0，那么向量 xxx 和向量 yyy 互相正交（orthogonal）。如果两个向量都有非零范数，那么这两个向量之间的夹角是 90∘90^{\circ}90∘。在 Rn\R^{n}Rn 中，至多有 nnn 个范数非零向量互相正交。如果这些向量不但互相正交，而且范数都为 111，那么我们称它们是标准正交（orthonormal）。

An orthogonal matrix is a square matrix whose rows are mutually orthonormal and whose columns are mutually orthonormal:

正交矩阵（orthogonal matrix）指行向量和列向量是分别标准正交的方阵，即

ATA=AAT=I.(2.37)A^{T}A=AA^{T}=I.\qquad\qquad\qquad(2.37)ATA=AAT=I.(2.37)

This implies that

这意味着

A−1=AT,(2.38)A^{−1}=A^{T},\qquad\qquad\qquad(2.38)A−1=AT,(2.38)

so orthogonal matrices are of interest because their inverse is very cheap to compute. Pay careful attention to the deﬁnition of orthogonal matrices. Counterintuitively, their rows are not merely orthogonal but fully orthonormal. There is no special term for a matrix whose rows or columns are orthogonal but not orthonormal.

正交矩阵受到关注是因为求逆计算代价小。我们需要注意正交矩阵的定义。违反直觉的是，正交矩阵的行向量不仅是正交的，还是标准正交的 。对于行向量或列向量互相正交但不是标准正交的矩阵，没有对应的专有术语。
