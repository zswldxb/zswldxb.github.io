2.1 Scalars, Vectors, Matrices and Tensors(标量、向量、矩阵和张量)

The study of linear algebra involves several types of mathematical objects:

学习线性代数，会涉及以下几个数学概念：

Scalars: A scalar is just a single number, in contrast to most of the other objects studied in linear algebra, which are usually arrays of multiple numbers. We write scalars in italics. We usually give scalars lowercase variable names. When we introduce them, we specify what kind of number they are. For example, we might say “Let s∈Rs\in\Rs∈R be the slope of the line,” while deﬁning a real-valued scalar, or “Let n∈Nn\in\Nn∈N be the number of units,” while deﬁning a natural number scalar.

标量（scalar）：一个标量就是一个单独的数，它不同于线性代数中研究的其他大部分对象（通常是多个数的数组）。我们用斜体表示标量。标量通常被赋予小写的变量名称。在介绍标量时，我们会明确它们是哪种类型的数。比如，在定义实数标量时，我们可能会说“令 s∈Rs\in\Rs∈R 表示一条线的斜率”；在定义自然数标量时，我们可能会说“令 n∈Nn\in\Nn∈N 表示元素的数目”。

Vectors : A vector is an array of numbers. The numbers are arranged in order. We can identify each individual number by its index in that ordering. Typically we give vectors lowercase names in bold typeface, such as xxx. The elements of the vector are identiﬁed by writing its name in italic typeface, with a subscript. The ﬁrst element of xxx is x1x_{1}x1​, the second element is x2x_{2}x2​, and so on. We also need to say what kind of numbers are stored in the vector. If each element is in R\RR, and the vector has nnn elements, then the vector lies in the set formed by taking the Cartesian product of R\RR nnn times, denoted as Rn\R^{n}Rn. When we need to explicitly identify the elements of a vector, we write them as a column enclosed in square brackets:

向量（vector）：一个向量是一列数。这些数是有序排列的。通过次序中的索引，我们可以确定每个单独的数。通常我们赋予向量粗体的小写变量名称，比如 xxx。向量中的元素可以通过带脚标的斜体表示。向量 xxx 的第一个元素是 x1x_{1}x1​，第二个元素是 x2x_{2}x2​，等等。我们也会注明存储在向量中的元素是什么类型的。如果每个元素都属于 R\RR，并且该向量有 nnn 个元素，那么该向量属于实数集 R\RR 的 nnn 次笛卡儿乘积构成的集合，记为 Rn\R^{n}Rn。当需要明确表示向量中的元素时，我们会将元素排列成一个方括号包围的纵列：

x=[x1x2⋮xn].(2.1)x = \begin{bmatrix} x_{1}\\ x_{2}\\ \vdots\\ x_{n} \end{bmatrix}.\qquad\qquad\qquad(2.1)x=⎣⎢⎢⎢⎡​x1​x2​⋮xn​​⎦⎥⎥⎥⎤​.(2.1)

We can think of vectors as identifying points in space, with each element giving the coordinate along a diﬀerent axis.

我们可以把向量看作空间中的点，每个元素是不同坐标轴上的坐标。

Sometimes we need to index a set of elements of a vector. In this case, we deﬁne a set containing the indices and write the set as a subscript. For example, to access x1x_{1}x1​, x3x_{3}x3​ and x6x_{6}x6​ , we deﬁne the set S={1，3，6}S=\lbrace1，3，6\rbraceS={1，3，6} and write xSx_{S}xS​. We use the −−− sign to index the complement of a set. For example x−1x_{−1}x−1​ is the vector containing all elements of xxx except for x1x_{1}x1​, and x−Sx_{−S}x−S​ is the vector containing all elements of xxx except for x1x_{1}x1​, x3x_{3}x3​ and x6x_{6}x6​.

有时我们需要索引向量中的一些元素。在这种情况下，我们定义一个包含这些元素索引的集合，然后将该集合写在脚标处。比如，指定 x1x_{1}x1​、x3x_{3}x3​ 和 x6x_{6}x6​，我们定义集合 S={1,3,6}S=\lbrace1,3,6\rbraceS={1,3,6}，然后写作 xSx_{S}xS​。我们用符号 −-− 表示集合的补集中的索引。比如 x−1x_{−1}x−1​ 表示 xxx 中除 x1x_{1}x1​ 外的所有元素，x−Sx_{−S}x−S​表示 xxx 中除 x1x_{1}x1​、x3x_{3}x3​、x6x_{6}x6​ 外所有元素构成的向量。

Matrices: AAA matrix is a 2-D array of numbers, so each element is identiﬁed by two indices instead of just one. We usually give matrices uppercase variable names with bold typeface, such as AAA. If a real-valued matrix AAA has a height of mmm and a width of nnn, then we say that A∈Rm×nA\in\R^{m\times n}A∈Rm×n. We usually identify the elements of a matrix using its name in italic but not bold font, and the indices are listed with separating commas. For example, A1,1A_{1,1}A1,1​ is the upper left entry of AAA and Am,nA_{m,n}Am,n​ is the bottom right entry. We can identify all the numbers with vertical coordinate iii by writing a “:” for the horizontal coordinate. For example, Ai,:A_{i,:}Ai,:​ denotes the horizontal cross section of AAA with vertical coordinate iii. This is known as the iii-th row of AAA. Likewise, A:,iA_{:,i}A:,i​ is the iii-th column of AAA. When we need to explicitly identify the elements of a matrix, we write them as an array enclosed in square brackets:

矩阵（matrix）：矩阵是一个二维数组，其中的每一个元素由两个索引（而非一个）所确定。我们通常会赋予矩阵粗体的大写变量名称，比如 AAA。如果一个实数矩阵高度为 mmm，宽度为 nnn，那么我们说 A∈Rm×nA\in\R^{m\times n}A∈Rm×n。我们在表示矩阵中的元素时，通常以不加粗的斜体形式使用其名称，索引用逗号间隔。比如，A1,1A_{1,1}A1,1​ 表示 AAA 左上的元素，Am,nA_{m,n}Am,n​ 表示 AAA 右下的元素。我们通过用 “:” 表示水平坐标，以表示垂直坐标 iii 中的所有元素。比如，Ai,:A_{i,:}Ai,:​ 表示 AAA 中垂直坐标 iii 上的一横排元素。这也被称为 AAA 的第 iii 行（row）。同样地，A:,iA_{:,i}A:,i​ 表示 AAA 的第 iii 列（column）。当需要明确表示矩阵中的元素时，我们将它们写在用方括号括起来的数组中：

[A1,1A1,2A2,1A2,2].(2.2)\begin{bmatrix} A_{1,1}&A_{1,2}\\ A_{2,1}&A_{2,2} \end{bmatrix}.\qquad\qquad\qquad(2.2)[A1,1​A2,1​​A1,2​A2,2​​].(2.2)

Sometimes we may need to index matrix-valued expressions that are not just a single letter. In this case, we use subscripts after the expression but do not convert anything to lowercase. For example, f(A)i,jf(A)_{i,j}f(A)i,j​ gives element (i,j)(i, j)(i,j) of the matrix computed by applying the function fff to AAA.

有时我们需要矩阵值表达式的索引，而不是单个元素。在这种情况下，我们在表达式后面接下标，但不必将矩阵的变量名称小写化。比如，f(A)i,jf(A)_{i,j}f(A)i,j​ 表示函数 fff 作用在 AAA 上输出的矩阵的第 iii 行第 jjj 列元素。

Tensors: In some cases we will need an array with more than two axes. In the general case, an array of numbers arranged on a regular grid with a variable number of axes is known as a tensor. We denote a tensor named “A” with this typeface: A\mathrm{A}A. We identify the element of A\mathrm{A}A at coordinates (i,j,k)(i, j, k)(i,j,k) by writing Ai,j,kA_{i,j,k}Ai,j,k​.

张量（tensor）：在某些情况下，我们会讨论坐标超过两维的数组。一般的，一个数组中的元素分布在若干维坐标的规则网格中，我们称之为张量。我们使用字体 A\mathrm{A}A 来表示张量“A”。张量 A\mathrm{A}A 中坐标为 (i,j,k)(i,j,k)(i,j,k) 的元素记作 Ai,j,kA_{i,j,k}Ai,j,k​。

One important operation on matrices is the transpose. The transpose of a matrix is the mirror image of the matrix across a diagonal line, called the main diagonal, running down and to the right, starting from its upper left corner. (AT)i,j=Aj,i.(2.3)(A^{T})_{i,j}= A_{j,i}.\qquad\qquad\qquad(2.3)(AT)i,j​=Aj,i​.(2.3)

Vectors can be thought of as matrices that contain only one column. The transpose of a vector is therefore a matrix with only one row. Sometimes we deﬁne a vector by writing out its elements in the text inline as a row matrix, then using the transpose operator to turn it into a standard column vector, for example x=[x1,x2,x3]Tx = [x_{1}, x_{2}, x_{3}]^{T}x=[x1​,x2​,x3​]T.

向量可以看作只有一列的矩阵。对应地，向量的转置可以看作只有一行的矩阵。有时，我们通过将向量元素作为行矩阵写在文本行中，然后使用转置操作将其变为标准的列向量，来定义一个向量，比如 x=[x1,x2,x3]Tx = [x_{1}, x_{2}, x_{3}]^{T}x=[x1​,x2​,x3​]T。

A scalar can be thought of as a matrix with only a single entry. From this, we can see that a scalar is its own transpose: a=aTa = a^{T}a=aT.

标量可以看作只有一个元素的矩阵。因此，标量的转置等于它本身，a=aTa = a^{T}a=aT。

We can add matrices to each other, as long as they have the same shape, just by adding their corresponding elements: C=A+BC = A + BC=A+B where Ci,j=Ai,j+Bi,jC_{i,j}= A_{i,j}+B_{i,j}Ci,j​=Ai,j​+Bi,j​.

只要矩阵的形状一样，我们可以把两个矩阵相加。两个矩阵相加是指对应位置的元素相加，比如 C=A+BC = A + BC=A+B，其中 Ci,j=Ai,j+Bi,jC_{i,j}= A_{i,j}+B_{i,j}Ci,j​=Ai,j​+Bi,j​。

We can also add a scalar to a matrix or multiply a matrix by a scalar, just by performing that operation on each element of a matrix: D=a⋅B+cD=a\cdot B+cD=a⋅B+c where Di,j=a⋅Bi,j+cD_{i,j}= a\cdot B_{i,j}+ cDi,j​=a⋅Bi,j​+c.

标量和矩阵相乘，或是和矩阵相加时，我们只需将其与矩阵的每个元素相乘或相加，比如 D=a⋅B+cD=a\cdot B+cD=a⋅B+c ，其中 Di,j=a⋅Bi,j+cD_{i,j}= a\cdot B_{i,j}+ cDi,j​=a⋅Bi,j​+c。

In the context of deep learning, we also use some less conventional notation. We allow the addition of a matrix and a vector, yielding another matrix: C=A+bC=A+bC=A+b, where Ci,j=Ai,j+bjC_{i,j}=A_{i,j}+b_{j}Ci,j​=Ai,j​+bj​. In other words, the vector bbb is added to each row of the matrix. This shorthand eliminates the need to deﬁne a matrix with bbb copied into each row before doing the addition. This implicit copying of bbb to many locations is called broadcasting.

在深度学习中，我们也使用一些不那么常规的符号。我们允许矩阵和向量相加，产生另一个矩阵：C=A+bC=A+bC=A+b，其中 Ci,j=Ai,j+bjC_{i,j}=A_{i,j}+b_{j}Ci,j​=Ai,j​+bj​。换言之，向量 bbb 和矩阵 AAA 的每一行相加。这个简写方法使我们无须在加法操作前定义一个将向量 bbb 复制到每一行而生成的矩阵。这种隐式地复制向量 bbb 到很多位置的方式，称为广播(broadcasting)。
