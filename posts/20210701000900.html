YOLOv3: An Incremental Improvement

Preface

Paper

Purpose

本文主要在 YOLOv2 的模型基础上进行了一点修改.

The Deal

Bounding Box Prediction

和 YOLO9000 一样, 我们使用通过 dimension clusters 得到的 anchor boxes 来预测 bounding boxes. 对于每个 bounding box, 网络预测 444 个坐标, tx,ty,tw,tht_{x},t_{y},t_{w},t_{h}tx​,ty​,tw​,th​. 如果 cell 相对于图片左上角的偏移量为 (cx,cy)(c_{x},c_{y})(cx​,cy​), anchor box 的宽度和高度为 pw,php_{w},p_{h}pw​,ph​, 则有

bx=σ(tx)+cxby=σ(ty)+cybw=pwetwbh=pheth\begin{aligned} b_{x}&=\sigma(t_{x})+c_{x}\\ b_{y}&=\sigma(t_{y})+c_{y}\\ b_{w}&=p_{w}e^{t_{w}}\\ b_{h}&=p_{h}e^{t_{h}}\\ \end{aligned}bx​by​bw​bh​​=σ(tx​)+cx​=σ(ty​)+cy​=pw​etw​=ph​eth​​

YOLOv3 采用 logistic regression 预测每个 bounding box 的 objectness score. 如果 bounding box prior 和某个 ground truth object 的 IOU 大于其他任意 bounding box prior, 则其值应为 1. 如果 bounding box prior 与某个 ground truth object 的 IOU 不是最大, 但是超过了阈值, 我们仍然忽略. 本文设置阈值 threshold 为 0.5.

Class Prediction

由于是 multi-label 分类, 因此将 softmax 分类器取代为 logistic 分类器, 本文使用二元交叉熵损失进行类预测.

Predictions Across Scales

YOLOv3 在三个不同的 scale 上预测 box. 本文模型类似金字塔网络.

Feature Extractor

本文将 YOLOv2, Darknet-19 和 residual network 相结合, 设计了 Darknet-53.
