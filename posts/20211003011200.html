Pytorch.torch.nn.SmoothL1Loss

损失函数层 SmoothL1Loss\text{SmoothL1Loss}SmoothL1Loss 为平滑的 L1\text{L1}L1 损失函数. 损失函数层 SmoothL1Loss\text{SmoothL1Loss}SmoothL1Loss 其要求标签张量 YYY, 输入张量 XXX 均为 nnn 维张量, 且 X,Y∈Rd1×⋯×dnX,Y\in\R^{d_{1}\times\cdots\times d_{n}}X,Y∈Rd1​×⋯×dn​.

reduction 参数

reduction: 用于指定计算的模式, 'none', 'mean' 或 'sum'. 默认为 'mean'.

当 reduction='none' 时, 计算过程如下:

beta 参数

beta 参数用于表示 L1\text{L1}L1 与 L2\text{L2}L2 损失的分界线. 默认为 1.0.

示例

import torch
import torch.nn as nn

print(f'torch version: {torch.__version__}')
# torch version: 1.10.2

# torch.set_printoptions(sci_mode=False)


def smooth_l1(input, target, reduction, beta):
    z = torch.abs(input - target)
    y_hat = torch.where(z < beta, z**2 / (2 * beta), z - 0.5 * beta)

    if reduction == 'mean':
        y_hat = torch.mean(y_hat)
    elif reduction == 'sum':
        y_hat = torch.sum(y_hat)

    return y_hat


def solve(features, reduction='mean', beta=1.0):
    reduction_list = ['none', 'mean', 'sum']
    assert reduction in reduction_list, 'mode error'

    input = torch.rand(features)
    target = (torch.rand(features) * 5).long().to(torch.float32)

    loss = nn.SmoothL1Loss(reduction=reduction, beta=beta)
    y = loss(input, target)

    y_hat = smooth_l1(input, target, reduction, beta)

    print(f'error: {(torch.abs(y-y_hat)).flatten()}')


if __name__ == '__main__':
    features = (10, 10)

    solve(features, reduction='none', beta=1.0)

    solve(features, reduction='mean', beta=1.0)

    solve(features, reduction='sum', beta=1.0)
