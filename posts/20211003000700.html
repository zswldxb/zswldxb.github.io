Pytorch.torch.nn.ConvTranspose1d

一维逆卷积与一维卷积类似, 可以结合 Pytorch.torch.nn.Conv1d 加深理解.

ConvTranspose1d\text{ConvTranspose1d}ConvTranspose1d 网络层为一维逆卷积层. 输入 Input\text{Input}Input 经过网络层 ConvTranspose1d\text{ConvTranspose1d}ConvTranspose1d 后得到的输出为 Output\text{Output}Output. 一维逆卷积层 ConvTranspose1d\text{ConvTranspose1d}ConvTranspose1d 要求其输入 Input\text{Input}Input 为三维张量, 且其维度含义依次为小批量维度, 输入通道维度和输入特征维度, 因此 Input∈Rbatch_size×in_channels×in_features\text{Input}\in\R^{\text{batch\_size}\times\text{in\_channels}\times\text{in\_features}}Input∈Rbatch_size×in_channels×in_features, 其输出 Output\text{Output}Output 为三维张量, 其维度含义依次为小批量维度, 输出通道维度和输出特征维度, 因此 Output∈Rbatch_size×out_channels×out_features\text{Output}\in\R^{\text{batch\_size}\times\text{out\_channels}\times\text{out\_features}}Output∈Rbatch_size×out_channels×out_features.

一维逆卷积层的相关参数

ConvTranspose1d\text{ConvTranspose1d}ConvTranspose1d 网络层的 __init__() 函数有多个参数, 不同的参数选择对应于不同的一维逆卷积网络层的变种.

  • in_features: 指定输入的通道数.
  • out_features: 指定输出的通道数.
  • kernel_size: 指定卷积核的大小.
  • stride: 指定步长大小.
  • output_padding: 用于调整输出的特征数.
  • padding: 指定填充大小.
  • dilation: 指定卷积核内每个元素的间隙.
  • groups: 指定输入通道到输出通道的阻塞连接数
  • bias: 指定是否使用偏置. True 为使用偏置, False 为不使用偏置.
  • padding_mode: 指定如何填充填充区的每个元素.
  • device: 指定模型所在的设备.

其中 in_features, out_features, kernel_size 在前面已经介绍过了. 接下来介绍bias, stride, output_padding, padding, dilation, groups 参数的用法.

bias 参数

bias 用于设置是否使用偏置项, True 为使用偏置, False 为不使用偏置, 默认为 True.

如图17所示. 对于三维张量输入 [[[5, 3, 6]]] 和三维张量卷积核 [[[7, 1]]], stride 为 333, output_padding为 222, padding 为 111, dilation 为 222. 一维逆卷积的过程为: 首先在输入特征维度的数据左右两侧分别填充 dilation×(kernel_size−1)=2\text{dilation}\times(\text{kernel\_size}-1)=2dilation×(kernel_size−1)=2 个 000, 在输入特征维度的每个元素之间填充 222 个 000, 在输入特征维度的数据右侧填充 222 个 000, 则输入变成 [[[0, 0, 5, 0, 0, 3, 0, 0, 6, 0, 0, 0, 0]]], 再在输入特征维度的左右两侧分别删除 111 个数据. 得到 ``[[[0, 5, 0, 0, 3, 0, 0, 6, 0, 0, 0]]], 然后将卷积核特征维度的数据反转, 则卷积核变为[[[1, 7]]], 然后按照一维卷积层的操作计算输出, 得到[[[0, 5, 21, 0, 3, 42, 0, 6, 0]]]`, 如其中 0=0×1+0×70=0\times1+0\times70=0×1+0×7, 5=5×1+0×75=5\times1+0\times75=5×1+0×7.

out_features=(in_features−1)×stride−2×padding+dilation×(kernel_size−1)+output_padding+1\text{out\_features}=(\text{in\_features}-1)\times\text{stride}-2\times\text{padding}+\text{dilation}\times(\text{kernel\_size}-1)+\text{output\_padding}+1out_features=(in_features−1)×stride−2×padding+dilation×(kernel_size−1)+output_padding+1

groups 参数

此前的 stride, output_padding, padding, dilation 参数作用于特征维度, 而不改变通道维度的计算方式, 而 groups 参数作用于通道维度, 而不改变特征维度的计算.

Pytorch 官方实现关于 groups 参数的介绍如下

例如当 groups=4 时, 对于输入维度为 (⋅,12,⋅)(\cdot,12,\cdot)(⋅,12,⋅), 输出维度为 (⋅,24,⋅)(\cdot,24,\cdot)(⋅,24,⋅). 此时按照通道维度将输入分为四组, 输出分为四组. 输入通道的四组的下标分别为 0-2,3-5,6-8,9-11, 输出通道的四组的下标分别为 0-5,6-11,12-17,18-23. 然后四组输入数据分别输入到四个不同的卷积核, 其输出通道数为 666, 然后得到四组输出, 然后将其按通道维度进行拼接.

值得注意的是, 设置 groups 时需要满足 in_features 和 out_features 都是 groups 的倍数.

当 in_features==groups 且 out_features==K*groups 时, 称其为深度可分离卷积.

如图18所示. 对于三维张量输入 [[[5, 3, 6], [3, 7, 2]]] 和三维张量卷积核 [[[7, 1]], [3, 4]], stride 为 333, output_padding为 222, padding 为 111, dilation 为 222, groups 为 222. 一维逆卷积的过程为: 首先在输入特征维度的数据左右两侧分别填充 222 个 000, 在输入特征维度的每个元素之间填充 222 个 000, 在输入特征维度的数据右侧填充 222 个 000, 则输入变成 [[[0, 0, 5, 0, 0, 3, 0, 0, 6, 0, 0, 0, 0], [0, 0, 3, 0, 0, 7, 0, 0, 2, 0, 0, 0, 0]]], 再在输入特征维度的左右两侧分别删除 111 个数据. 得到 ``[[[0, 5, 0, 0, 3, 0, 0, 6, 0, 0, 0], [0, 3, 0, 0, 7, 0, 0, 2, 0, 0, 0]]], 然后将卷积核特征维度的数据反转, 则卷积核变为[[[1, 7]], [[4, 3]]], 然后按照一维卷积层的操作计算输出, 得到[[[0, 5, 21, 0, 3, 42, 0, 6, 0], [0, 12, 21, 0, 28, 6, 0, 8, 0]]]`, 如其中 0=0×1+0×70=0\times1+0\times70=0×1+0×7, 0=0×3+0×40=0\times3+0\times40=0×3+0×4.
