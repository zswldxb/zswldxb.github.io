SSD: Single Shot MultiBox Detector

Preface

Paper

Purpose

本文为 one stage 的多尺度的端到端的目标检测模型.

The Single Shot Detector (SSD)

Feature map cell, default box, scale 和 aspect ratio

Feature map cell 是指 feature map 中每一个小格子, 如对于宽度和高度分别为 888 和 888 的输出特征图, 其 feature map cell 的个数为 646464. 本文将在多个尺度上进行学习. 将对不同宽高的特征图进行检测. 由于不同尺度下的 feature map cell 在原图中的感受野不同(相差一个倍数), 因此可以检测不同大小的物体.

Default box 是指每个 feature map cell 上的一系列 box. 如每个 cell 上有 444 个 box.

此外对于每个 default box 都需要预测 ccc 个 score 和 444 个 offset. 本文设置一个分类器, 用于预测每个 default box 的类别, 设置一个回归器, 用于预测每个 default box 的 offset.

因此, 本文对于每个宽高为 m×nm\times nm×n 的输出特征图, 设置 kkk 个 default box, 则该特征图有 m×nm\times nm×n 个 feature map cell. 设置一个 3×33\times 33×3 的卷积核作为分类器, 此卷积核的输出维度为 k×ck\times ck×c, 设置另一个 3×33\times 33×3 的卷积核作为回归器, 此卷积核的输出维度为 k×4k\times 4k×4.

对于每一个 default box, 需要设置其 scale 和 aspect ratio.

对于每个输出特征图, 设置其 default box 的 scale 为:

可以看出 default box 在不同的 feature 层有不同的 scale，在同一个 feature 层又有不同的 aspect ratio，因此基本上可以覆盖输入图像中的各种形状和大小的 object.

Model

本文首先通过深度卷积网络结构提取特征, 分别得到不同尺度下的特征表示. 然后对不同尺度上的特征图按上面的方式得到 box, 并训练网络.

Training

在训练期间, 需要将 default box 与 ground truth box 对应, 策略如下: 首先将每个 ground truth box 与具有最佳重叠的 default box 相匹配, 然后对剩下的 default box, 将其与 IoU≥0.5IoU\geq 0.5IoU≥0.5 的 ground truth box 匹配. 这些匹配的 default box 为候选正样本集. 由于负样本集的 box 远多于正样本集的 box. 这会导致样本不均衡. 训练难以收敛. 因此本文采用一定的策略控制正负样本比例为 1:31:31:3.
