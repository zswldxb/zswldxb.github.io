重复组合

从 nnn 个不同元素中每次取出一个，放回后再取下一个，如此连续取 rrr 次所得的组合称为重复组合，其组合总数记作 HnrH^{r}_{n}Hnr​，其数值为 (n+r−1r)\displaystyle{\binom{n+r-1}{r}}(rn+r−1​)。注意：这里的 rrr 也允许大于 nnn。

证明

证法 111：

设有 nnn 个不同的元素，不失一般性，可设为 1,2,⋯ ,n1,2,\cdots,n1,2,⋯,n。

设从这 nnn 个不同的元素取出 rrr 个元素的重复组合为

x1,x2,⋯ ,xr,(1)\begin{aligned} &x_{1},x_{2},\cdots,x_{r},&(1) \end{aligned}​x1​,x2​,⋯,xr​,​(1)​

且假定 x1≤x2≤⋯≤xrx_{1}\leq x_{2}\leq \cdots \leq x_{r}x1​≤x2​≤⋯≤xr​，这里有等号出现是因为元素允许重复。

我们再构造一个组合

y1,y2,⋯ ,yr,(2)\begin{aligned} &y_{1},y_{2},\cdots,y_{r},&(2) \end{aligned}​y1​,y2​,⋯,yr​,​(2)​

与组合 (1)(1)(1) 相对应，其对应关系是

yk=xk+k−1,(1≤k≤r)y_{k}=x_{k}+k-1,(1\leq k\leq r)yk​=xk​+k−1,(1≤k≤r)

显然这种对应是一一对应。在这个对应中，虽然组合 (1)(1)(1) 中的元素有可能相同，但是组合 (2)(2)(2) 中的元素却都不相同，这样，组合 (2)(2)(2) 就是一个没有重复元素的组合。

由于 y1y_{1}y1​ 最小可以取 111，yry_{r}yr​ 最大可以取 n+r−1n+r-1n+r−1，yky_{k}yk​ 的取值范围为 k≤yk≤n+k−1k\leq y_{k}\leq n+k-1k≤yk​≤n+k−1。

所以组合 (2)(2)(2) 是从 n+r−1n+r-1n+r−1 个不同的元素中，取出 rrr 个不同元素的组合，组合数为 (n+r−1r)\displaystyle{\binom{n+r-1}{r}}(rn+r−1​)。由于组合 (2)(2)(2) 的组合数与组合 (1)(1)(1) 的组合数相同，所以组合 (1)(1)(1) 的组合数为 Hnr=(n+r−1r)H^{r}_{n}=\displaystyle{\binom{n+r-1}{r}}Hnr​=(rn+r−1​)。

证法 222：

将此 nnn 个元素画成 nnn 个盒子（用 n+1n+1n+1 个 111 表示），如果第 iii 个元素取到过 111 次，则在此盒子中用 000 作 111 次记号。如 1001101⋯100011001101\cdots 100011001101⋯10001，表示第 111 个元素取到个 222 次，第二个元素取到过 000 次，第 333 个元素取到过 111 次，⋯\cdots⋯，第 nnn 个元素取到过 333 次，因为共取 rrr 次，所以共有 rrr 个 000，n+1n+1n+1 个 111，如此所有的 rrr 个 000 和 n+1n+1n+1 个 111 中除了两端的那两个 111 不可以动之外，共有 n+r−1n+r-1n+r−1 个 000 和 111 可随意放置，不同的放置表示不同的取法。因此重复组合数就等于在次 n+r−1n+r-1n+r−1 个位置上任选 rrr 个放 000，或在此 n+r−1n+r-1n+r−1 个位置上任选 n−1n-1n−1 个放 111，而 (n+r−1r)=(n+r−1n−1)\displaystyle{\binom{n+r-1}{r}}=\displaystyle{\binom{n+r-1}{n-1}}(rn+r−1​)=(n−1n+r−1​)，所以 Hnr=(n+r−1r)H^{r}_{n}=\displaystyle{\binom{n+r-1}{r}}Hnr​=(rn+r−1​)。

证法 333：

定理 111：一次不定方程 x1+x2+⋯+xn=r,(n≤r)x_{1}+x_{2}+\cdots+x_{n}=r,(n\leq r)x1​+x2​+⋯+xn​=r,(n≤r) 的任一组正整数解的个数为 (r−1n−1)\displaystyle{\binom{r-1}{n-1}}(n−1r−1​)。

证明定理 111：

一次不定方程的任一组正整数解的个数等同于在 nnn 个盒子中放 rrr 个球。用 rrr 个 000 表示 rrr 个球，用 n+1n+1n+1 个 111 表示 nnn 个盒子，则先将 rrr 个球摆成一排 0⋯00\cdots 00⋯0，然后在左右两边分别摆上 111，为 10⋯0110\cdots0110⋯01，然后再剩下的 000 与 000 之间插入 n−1n-1n−1 个 111，由于 000 与 000 之间的空位还有 r−1r-1r−1 个，所以一次不定方程 x1+x2+⋯+xn=r,(n≤r)x_{1}+x_{2}+\cdots+x_{n}=r,(n\leq r)x1​+x2​+⋯+xn​=r,(n≤r) 的任一组正整数解的个数为 (r−1n−1)\displaystyle{\binom{r-1}{n-1}}(n−1r−1​)。

定理 222：一次不定方程 x1+x2+⋯+xn=rx_{1}+x_{2}+\cdots+x_{n}=rx1​+x2​+⋯+xn​=r 的任一组非负整数解的个数为 (n+r−1n−1)\displaystyle{\binom{n+r-1}{n-1}}(n−1n+r−1​)。

证明定理 222：

设 yi=xi+1,1≤i≤ny_{i}=x_{i}+1,1\leq i\leq nyi​=xi​+1,1≤i≤n，则一次不定方程 x1+x2+⋯+xn=rx_{1}+x_{2}+\cdots+x_{n}=rx1​+x2​+⋯+xn​=r 的任一组非负整数解的个数，等价于一次不定方程 (y1−1)+(y2−1)+⋯+(yn−1)=r(y_{1}-1)+(y_{2}-1)+\cdots+(y_{n}-1)=r(y1​−1)+(y2​−1)+⋯+(yn​−1)=r 的任一组正整数解的个数，即一次不定方程 y1+y2+⋯+yn=n+ry_{1}+y_{2}+\cdots+y_{n}=n+ry1​+y2​+⋯+yn​=n+r 的任一组正整数解的个数，为 (n+r−1n−1)=(n+r−1r)\displaystyle{\binom{n+r-1}{n-1}=\binom{n+r-1}{r}}(n−1n+r−1​)=(rn+r−1​)。

重复组合个数证明：

设有 nnn 个不同的元素，不失一般性，可设为 1,2,⋯ ,n1,2,\cdots,n1,2,⋯,n。

从 nnn 个不同的元素取出 rrr 个元素的重复组合为 i1,i2,⋯ ,iri_{1},i_{2},\cdots,i_{r}i1​,i2​,⋯,ir​。

设元素 k∈{1,2,⋯ ,n}k\in\{1,2,\cdots,n\}k∈{1,2,⋯,n} 在组合中出现了 xkx_{k}xk​ 次，其中 xkx_{k}xk​ 是非负整数，若 xk=0x_{k}=0xk​=0，则说明元素 kkk 在组合中没有出现；若 xk=3x_{k}=3xk​=3，则说明元素 kkk 在组合中出现 333 次，由此，一次不定方程 x1+x2+⋯+xn=rx_{1}+x_{2}+\cdots+x_{n}=rx1​+x2​+⋯+xn​=r 的任一组非负整数解就对应着一个 rrr 个元素的重复组合。所以，从 nnn 个不同的元素取出 rrr 个元素的重复组合数，就是一次不定方程 x1+x2+⋯+xn=rx_{1}+x_{2}+\cdots+x_{n}=rx1​+x2​+⋯+xn​=r 的任一组非负整数解的个数，其个数为 (n+r−1n−1)\displaystyle{\binom{n+r-1}{n-1}}(n−1n+r−1​)，即 Hnr=(n+r−1r)H^{r}_{n}=\displaystyle{\binom{n+r-1}{r}}Hnr​=(rn+r−1​)。
