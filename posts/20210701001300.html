Densely connected convolutional networks

Preface

Paper

Purpose

本文提出了 DenseNet, 将每一层都连接到其他每一层, 以缓解梯度消失的问题, 增强特征传播, 鼓励特征重用.

DenseNet

本文的网络结构示意图如下所示:

假设某个卷积神经网络的层数为 LLL, 每层实现一个非线性变换 Hl(⋅)H_{l}(\cdot)Hl​(⋅), lll 是层的编号, Hl(⋅)H_{l}(\cdot)Hl​(⋅) 可以是 Batch Normalization(BN), rectified linear units(ReLU), Pooling, 或者 Convolution(Conv). 第 lll 层 的输出为 xlx_{l}xl​.

ResNets

传统卷积的第 lll 层的输出为 xl=Hl(xl−1)x_{l}=H_{l}(x_{l-1})xl​=Hl​(xl−1​). ResNet 增加了一个 identity function, 第 lll 层的输出为

xl=Hl(xl−1)+xl−1.x_{l}=H_{l}(x_{l-1})+x_{l-1}.xl​=Hl​(xl−1​)+xl−1​.

这使得 ResNet 模型的梯度可以直接通过 identity function 从后面的层流向前面的层. 然而 identity function 和 HlH_{l}Hl​ 的输出直接相加, 这可能阻碍网络的信息传递.

Dense connectivity

本文提出了不同的连接模式: 本文引入了从任何层到后续所有层的直接连接. 因此, 第 lll 层接收所有前面的层 x0,x1,⋯ ,xl−1x_{0},x_{1},\cdots,x_{l-1}x0​,x1​,⋯,xl−1​ 的特征图作为输入:

xl=Hl([x0,x1,⋯ ,xl−1])x_{l}=H_{l}([x_{0},x_{1},\cdots,x_{l-1}])xl​=Hl​([x0​,x1​,⋯,xl−1​])

其中 [x0,x1,⋯ ,xl−1][x_{0},x_{1},\cdots,x_{l-1}][x0​,x1​,⋯,xl−1​] 是指层 0,⋯ ,l−10,\cdots,l-10,⋯,l−1 中的输出特征图在通道维度上拼接后的张量.

Composite function

本文定义的 Hl(⋅)H_{l}(\cdot)Hl​(⋅) 为三个连续操作的复合函数: batch normalization(BN), rectified linear unit(ReLU) 和 3×33\times33×3 convolutional layer(Conv).

Pooling layers

由于本文定义的 dense block 结构在特征图的宽度和高度发生改变时不可行, 然而 down-sampling 操作十分重要且会改变特征图的宽度和高度, 为了便于本文网络使用 down-sampling, 本文将网络分为多个密集连接的 dense blocks 和在 dense blocks 之间的 transition layers, 本文的 transition 由 batch normalization(BN), 1×11\times 11×1 convolutional layer 和 2×22\times 22×2 average pooling layer 组成.

Growth rate

假设每个函数 HlH_{l}Hl​ 产生通道数为 kkk 个的特征图, 则 lll 层的输入特征图的通道数为 k0+k×(l−1)k_{0}+k\times (l-1)k0​+k×(l−1) 个输入特征图, k0k_{0}k0​ 为输入层的通道数.

Bottleneck layers

尽管每层只有 kkk 个输出特征图, 然而其输入通道数很多. 本文在每次进行 3×33\times33×3 卷积之前使用 1×11\times 11×1 卷积作为 Bottleneck layer. 因此本文的 HlH_{l}Hl​ 修改为 BN-ReLU-Conv(1×11\times11×1)-BN-ReLU-Conv(3×33\times33×3), 并将使用方式的模型称为 DenseNet-B. 本文设置 1×11\times 11×1 Conv 的通道数为 4k4k4k.

Compression

为了提高模型的紧凑型, 减少 transition layer 的特征图数量, 本文设置压缩因子 θ\thetaθ, 如果一个 dense block 的输出特征图的通道数为 mmm, 则过渡层的输出特征图的通道数为 ⌊θm⌋\lfloor\theta m\rfloor⌊θm⌋(0<θ≤10<\theta\leq 10<θ≤1). 本文将 θ<1\theta<1θ<1 的 DenseNet 模型称为 DenseNet-C, 并且本文设置 θ=0.5\theta=0.5θ=0.5, 称同时使用 bottleneck 和 θ<1\theta<1θ<1 的 transition layer 的模型为 DenseNet-BC.
