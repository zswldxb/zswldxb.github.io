Pytorch.torch.nn.EmbeddingBag

网络层 Embedding\text{Embedding}Embedding 为嵌入层. 对于文本数据, 我们会设置一个字典, 将大多数出现的单词放入字典中, 此外还加入一些特殊字符表示特殊含义, 并且给每个单词一个编号, 设字典中单词的总数为 MMM. 在需要计算时, 我们使用编号(数字)来代替单词(字符串), 这样计算机便可以进行数值计算. 但是由于编号的差并不能表示对应单词之间的距离, 所以提出了 one-hot 编码, 对于每一个单词, 设置一个长度为 MMM 向量, 并将向量下标为这个单词对应的编号的位置置为 111, 其余位置为 000. 这样在一个文本中相同的单词对应的 one-hot 编码相同, 不同单词对应的 one-hot 编码不同. 但是由于字典长度 MMM 通常很大, 因此 one-hot 编码所表示的空间为 2M2^{M}2M 是巨大的, 而 one-hot 编码仅会使用一小部分空间, 造成了存储和计算的巨大浪费, 而且相似的两个单词之间的距离与不相似的两个单词之间的距离是相同的. 因此需要采用网络层 Embedding\text{Embedding}Embedding 来将 one-hot 编码进行压缩. 而这个过程等价于将一个单词的编号直接映射到网络层 Embedding\text{Embedding}Embedding 对应的空间中.

num_embeddings 参数

num_embedding 参数用于指定字典的大小.

embedding_dim 参数

embedding_dim 参数用于指定每个嵌入向量的大小.

padding_idx 参数

padding_idx 参数用于指定"填充"的编号.

max_norm 参数

max_norm 参数用于指定嵌入向量的最大范数值, 当超过这一数值时, 将被替换为 max_norm.

norm_type 参数

norm_type 参数用于指定计算范数的方式.

scale_grad_by_freq 参数

scale_grad_by_freq 参数用于指定是否根据每个单词出现的频率放缩梯度. 在 mode='max' 时不使用.

mode 参数

mode 参数可为 'sum', 'mean', 'max'.

mode='sum' 等价于 torch.nn.Embedding + torch.sum(dim=1).

mode='mean' 等价于 torch.nn.Embedding + torch.mean(dim=1).

mode='max' 等价于 torch.nn.Embedding + torch.max(dim=1).

sparse 参数

sparse 参数用于指定嵌入向量是否为稀疏张量.

示例

import torch
import torch.nn as nn

print(f'torch version :{torch.__version__}')

torch.manual_seed(1)


def solve(features,
          num_embeddings,
          embedding_dim,
          padding_idx=0,
          max_norm=False,
          norm_type=2,
          scale_grad_by_freq=False,
          sparse=False):
    input = (torch.rand(features) * num_embeddings).long()
    embedding = nn.Embedding(num_embeddings=num_embeddings,
                             embedding_dim=embedding_dim,
                             padding_idx=padding_idx,
                             max_norm=max_norm,
                             norm_type=norm_type,
                             scale_grad_by_freq=scale_grad_by_freq,
                             sparse=sparse)

    embeddingbag = nn.EmbeddingBag(num_embeddings=num_embeddings,
                                   embedding_dim=embedding_dim,
                                   padding_idx=padding_idx,
                                   max_norm=max_norm,
                                   norm_type=norm_type,
                                   mode='sum',
                                   scale_grad_by_freq=scale_grad_by_freq,
                                   sparse=sparse)


    output1 = embedding(input)
    output2 = embeddingbag(input)

    print(output1.shape)
    print(output2.shape)

    print(output1.sum(dim=1) - output2)

if __name__ == '__main__':
    features = (5, 8)

    solve(features, num_embeddings=10000, embedding_dim=100)
