Pytorch.torch.nn.BCEWithLogitsLoss

损失函数层 BCEWithLogitsLoss\text{BCEWithLogitsLoss}BCEWithLogitsLoss 为带逻辑回归函数的二分类交叉熵损失函数. 损失函数层 BCEWithLogitsLoss\text{BCEWithLogitsLoss}BCEWithLogitsLoss 其要求标签张量 YYY, 输入张量 XXX 均为 nnn 维张量, 且 X,Y∈Rd1×⋯×dnX,Y\in\R^{d_{1}\times\cdots\times d_{n}}X,Y∈Rd1​×⋯×dn​. 且 YYY 的每个元素取值集合为 {0,1}\{0,1\}{0,1}.

对于标签为 yyy, 预测为 xxx, 二分类交叉熵定义如下:

l(x,y)=−(ylog⁡(x)+(1−y)log⁡(1−x))l(x,y)=-(y\log {(x)}+(1-y)\log{(1-x)})l(x,y)=−(ylog(x)+(1−y)log(1−x))

这里以 X,Y∈RN×CX,Y\in\R^{N\times C}X,Y∈RN×C, 其中 NNN 表示样本数, CCC 表示标签个数.

weight 参数

weight 参数用于加权每个元素的权重. W∈RN×CW\in\R^{N\times C}W∈RN×C.

pos_weight 参数

pos_weight 参数用于表示需要对正样本的加权比例, 其计算公式为 反例数正例数\dfrac{\text{反例数}}{\text{正例数}}正例数反例数​. U∈RCU\in\R^{C}U∈RC

reduction 参数

reduction: 用于指定计算的模式, 'none', 'mean', 'sum' 或 'batchmean'. 默认为 'mean'.

在使用 weight 参数和 pos_weight 参数, 当 reduction='none' 时, 计算过程如下:

示例

import torch
import torch.nn as nn

print(f'torch version: {torch.__version__}')
# torch version: 1.10.2

# torch.set_printoptions(sci_mode=False)


def bce_logits(input, target, weight, reduction, pos_weight):
    y_hat = weight * (((pos_weight - 1) * target + 1) * torch.log(1 + torch.exp(-input)) + input * (1 - target))

    if reduction == 'mean':
        y_hat = torch.mean(y_hat)
    elif reduction == 'sum':
        y_hat = torch.sum(y_hat)

    return y_hat


def solve(batch_size, num_class, reduction='mean', use_pos_weight=False):
    reduction_list = ['none', 'mean', 'sum']
    assert reduction in reduction_list, 'mode error'

    input = torch.rand(batch_size, num_class)
    target = (torch.rand(batch_size, num_class) * 2).long().to(torch.float32)
    weight = torch.rand(batch_size, num_class)
    pos_weight = torch.rand(num_class) if use_pos_weight else torch.ones(num_class)

    loss = nn.BCEWithLogitsLoss(weight=weight, reduction=reduction, pos_weight=pos_weight)
    y = loss(input, target)

    y_hat = bce_logits(input, target, weight, reduction, pos_weight)

    print(f'error: {(torch.abs(y-y_hat)).flatten()}')


if __name__ == '__main__':
    batch_size, num_class = 5, 10

    solve(batch_size, num_class, reduction='none', use_pos_weight=False)

    solve(batch_size, num_class, reduction='mean', use_pos_weight=False)

    solve(batch_size, num_class, reduction='sum', use_pos_weight=False)

    solve(batch_size, num_class, reduction='none', use_pos_weight=True)

    solve(batch_size, num_class, reduction='mean', use_pos_weight=True)

    solve(batch_size, num_class, reduction='sum', use_pos_weight=True)
