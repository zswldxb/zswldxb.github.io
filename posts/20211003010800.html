Pytorch.torch.nn.CrossEntropyLoss

损失函数层 CrossEntropyLoss\text{CrossEntropyLoss}CrossEntropyLoss 为交叉熵损失函数.

损失函数层 CrossEntropyLoss\text{CrossEntropyLoss}CrossEntropyLoss 其参数如下:

  • weight: 为加权矩阵, 用于对每个类别进行加权处理.
  • reduction: 用于指定输出的模式, 'none', 'mean' 或 'sum'. 默认为 'mean'.
  • ignore_index: 表示忽略计算交叉熵的类别, 并且不会计算其梯度.
  • label_smoothing: 用于平滑操作, 降低过拟合的可能性.

weight 参数

weight 参数用于指定对每个类的加权系数. 默认不使用权重. 该参数可以用于缓解类不平衡的问题. 权重矩阵为 W∈RCW\in\R^{C}W∈RC, 不采用权重参数时可视其为全 111 张量.

reduction 参数

reduction: 用于指定输出的模式, 'none', 'mean' 或 'sum'. 默认为 'mean'.

ignore_index 参数

ignore_index: 表示忽略计算交叉熵的类别, 并且不会计算其梯度.

在使用 weight 参数, ignore_index 参数, reduction='mean' 时. 由于 Pytorch 要求使用 ignore_index 参数时, 真实张量为 n+1n+1n+1 维张量 Y∈RN×d1×⋯×dnY\in\R^{N\times d_{1}\times\cdots\times d_{n}}Y∈RN×d1​×⋯×dn​, 其中每个元素表示样本在指定特征下的类别(每个元素指示一个类别), 预测张量为 n+2n+2n+2 维张量 X∈RN×C×d1×⋯×dnX\in\R^{N\times C\times d_{1}\times\cdots\times d_{n}}X∈RN×C×d1​×⋯×dn​, 权重张量 W∈RCW\in\R^{C}W∈RC.

由于需要忽略类别为 ignore_index\text{ignore\_index}ignore_index 的所有样本, 因此将 Wignore_indexW_{\text{ignore\_index}}Wignore_index​ 置为零得到 W′W^{'}W′. 因此交叉熵的计算公式如下:

示例

import torch
import torch.nn as nn

print(f'torch version: {torch.__version__}')
# torch version: 1.10.2

# torch.set_printoptions(sci_mode=False)


def cross_entropy(input, target, weight, is_same_shape):
    x_exp = input.exp()
    z = weight * torch.log(x_exp / torch.sum(x_exp, dim=1, keepdim=True))
    if is_same_shape: # n+2 dim input and n+2 dim target
        y_hat = - torch.sum(target * z, dim=1)
    else: # n+2 dim input and n+1 dim target
        y_hat = - (torch.gather(z, dim=1, index=target.unsqueeze(1))).squeeze(1)
    return y_hat

def solve(batch_size, channels, features, is_same_shape=True, use_weight=False, mode='mean', ignore_index=-100):
    mode_list = ['none', 'mean', 'sum']
    assert mode in mode_list, 'mode error'
    ignore_index_mode = 0 <= ignore_index and ignore_index < channels
    is_same_shape = (not ignore_index_mode) and is_same_shape

    input = torch.rand((batch_size, channels, *features))
    weight = torch.rand(channels) if use_weight else torch.ones(channels)

    if is_same_shape: # n+2 dim input and n+2 dim target
        target = torch.rand((batch_size, channels, *features))
    else: # n+2 dim input and n+1 dim target
        target = (torch.rand((batch_size, *features)) * channels).long()
        one_hot = torch.zeros(batch_size, channels, *features).scatter_(dim=1, index=target.unsqueeze(1), value=1)
    # print(target)

    if ignore_index_mode:
        weight[ignore_index] = 0.0

    loss = nn.CrossEntropyLoss(reduction=mode, weight=weight, ignore_index=ignore_index)
    y = loss(input, target)

    weight = weight.reshape(1, channels, *[1 for _ in range(len(features))])
    y_hat = cross_entropy(input, target, weight, is_same_shape)

    if mode == 'mean':
        if is_same_shape: # n+2 dim input and n+2 dim target
            y_hat = torch.mean(y_hat)
        else: # n+2 dim input and n+1 dim target
            y_hat = torch.sum(y_hat) / torch.sum(weight * one_hot)
    elif mode == 'sum':
        y_hat = torch.sum(y_hat)

    print(f'error: {(torch.abs(y-y_hat)).flatten()}')


if __name__ == '__main__':
    batch_size = 2
    channels = 8
    features = (3, 4)
    solve(batch_size, channels, features)

    solve(batch_size, channels, features, is_same_shape = True, use_weight=True, mode='none')

    solve(batch_size, channels, features, is_same_shape = True, use_weight=True, mode='mean')

    solve(batch_size, channels, features, is_same_shape = True, use_weight=True, mode='sum')

    solve(batch_size, channels, features, is_same_shape = False, use_weight=True, mode='none')

    solve(batch_size, channels, features, is_same_shape = False, use_weight=True, mode='mean')

    solve(batch_size, channels, features, is_same_shape = False, use_weight=True, mode='sum')

    solve(batch_size, channels, features, mode='none', ignore_index=0)

    solve(batch_size, channels, features, mode='mean', ignore_index=0)

    solve(batch_size, channels, features, mode='sum', ignore_index=0)
