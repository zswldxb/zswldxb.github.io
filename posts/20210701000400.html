Rich feature hierarchies for accurate object detection and semantic segmentation

Preface

Paper

Purpose

本文为目标检测的开山之作. 本文将 region proposal 与 CNN 相结合, 提出了 R-CNN 模型.

Methodology

Framework

R-CNN 由三个模块组成:

  1. 生成 category-independent 的 region proposal. 这些 proposal 构成了检测器的候选检测集.
  2. 大型卷积神经网络. 对于每个 region, 它都提取一个固定长度的特征向量.
  3. 一组对应于特定类别的线性 SVM.

Module design

Region proposals. 目前已有很多 category-independent 的 region proposal: objectness [1], selective search [2], category-independent object proposals [3], constrained parametric min-cuts (CPMC) [4], multi-scale combinatorial grouping [5], and Ciresan et al. [6]. 本文采用 selective search.

Feature extraction. 本文采用 the CNN described by Krizhevsky et
al.[7]. 本文对于任意大小的 region, 都将其直接 resize 为 227×227227\times 227227×227(因为 CNN 的输入大小为 227×227227\times 227227×227). 在 resize 之前对边缘填充 ppp 个像素(本文采用 p=16p=16p=16).

Test-time detection. 在测试时, 首先使用 selective search 对每个图像分别提取 200020002000 个 region proposal, 然后 resize proposal 的大小为 CNN 的输入大小, 并经过 CNN 训练得到 feature map, 并将 feature map 输入一组 SVM, 对于每个类别, 其对应的 SVM 给出评分. 由于每个图像的 region proposal 过多, 本文采用 greedy non-maximun suppression(独立地针对每个类别), 这样可以减少 region proposal 的数量.

Training

本文采用迁移学习的思想, 首先在 ILSVRC2012 classification 上训练, 然后将训练好的模型的分类器替换为 N+1N+1N+1 路分类器(NNN 路为 object class 的总数, 111 路为背景), 并进行微调.

Object category classifiers. 本文采用 IoU 重叠来判断是否为 object class, 如果 IoU 大于设定的 IoU 阈值, 则为正例, 否则为反例. 本文将一组 SVM 作为分类器.

Bounding-box regression

本文其余内容在这不做介绍, 仅介绍对后续文章有重要影响的 Bounding-box regression.

本文提出了一种减少定位误差的方法, 称为 Bounding-box regression.
