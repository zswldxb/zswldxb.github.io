Motivation

在 NLP 中, 句子由若干个单词组成, 本文将图片视为有视觉句子组成, 而视觉句子由视觉单词组成, 将图片分为若干个 patch, 再将每个 patch 分为若干个 sub-patch. 然后计算视觉单词的注意力和视觉句子的注意力.

Preliminaries

MSA. 对于自注意力模块, 假定输入为 X∈Rn×dX\in\R^{n\times d}X∈Rn×d, 将其分别经过三个全连接层得到 query Q∈Rn×dkQ\in\R^{n\times d_{k}}Q∈Rn×dk​, key K∈Rn×dkK\in\R^{n\times d_{k}}K∈Rn×dk​, value V∈Rn×dvV\in\R^{n\times d_{v}}V∈Rn×dv​, 其中 nnn 为序列长度, ddd, dkd_{k}dk​, dvd_{v}dv​ 为输入, query(``key),value` 的特征维度.

然后经过 scaled dot-product attention 计算注意力, scaled dot-product attention 计算公式为:

Attention(Q,K,V)=softmax(QKTdk)VAttention(Q,K,V)=softmax(\dfrac{QK^{T}}{\sqrt{d_{k}}})VAttention(Q,K,V)=softmax(dk​​QKT​)V

将注意力的结果经过全连接层得到输出.

多头注意力模块则是将 query, key, value 沿特征维度分为 hhh 个部分, 然后每个部分计算注意力, 然后将这 hhh 个注意力沿特征维度拼接, 并经过全连接层得到输出.

MLP. MLP 层应用于自注意力模块之后, 其用于增加非线性, 假设输入为 X∈Rn×dX\in\R^{n\times d}X∈Rn×d, MLP 层计算公式如下:

MLP(X)=FC(σ(FC(X))),FC(X)=XW+bMLP(X)=FC(\sigma(FC(X))),\qquad FC(X)=XW+bMLP(X)=FC(σ(FC(X))),FC(X)=XW+b

其中 WWW 和 bbb 是全连接层的权重和偏移, σ(⋅)\sigma(\cdot)σ(⋅) 为激活函数, 通常采用 GELU\text{GELU}GELU.

LN. Layer normalization 为归一化层, 对于输入为 x∈Rdx\in\R^{d}x∈Rd, 其计算公式如下:

LN(x)=x−μσ∘γ+βLN(x)=\frac{x-\mu}{\sigma}\circ \gamma +\betaLN(x)=σx−μ​∘γ+β

其中 μ∈R\mu\in\Rμ∈R, δ∈R\delta\in\Rδ∈R 为特征的均值和标准差, ∘\circ∘ 为按位乘法, γ∈Rd\gamma\in\R^{d}γ∈Rd, β∈Rd\beta\in\R^{d}β∈Rd 为可学习参数.

Transformer in Transformer

对于给定的输入, 将其均匀的划分为 nnn 个 patch X=[X1,X2,⋯ ,Xn]∈Rn×p×p×3\mathcal{X}=[X^{1},X^{2},\cdots,X_{n}]\in\R^{n\times p \times p \times 3}X=[X1,X2,⋯,Xn​]∈Rn×p×p×3, 其中 (p,p)(p,p)(p,p) 是每个 patch 的分辨率. 在本文中, 将每个 patch 进一步均匀分为 mmm 个 sub-patch. 在 NLP 中, 句子由若干个单词组成, 本文将 patch 视为句子, sub-patch 视为单词.

Xi→[xi,1,xi,2,⋯ ,xi,m]X^{i}\rightarrow [x^{i,1},x^{i,2},\cdots,x^{i,m}]Xi→[xi,1,xi,2,⋯,xi,m]

其中 xi,j∈Rs×s×3x^{i,j}\in\R^{s\times s\times 3}xi,j∈Rs×s×3 表示第 iii 个视觉句子的第 jjj 个视觉单词, (s,s)(s,s)(s,s) 是每个 sub-patch 的空间大小, j=1,2,⋯ ,mj=1,2,\cdots,mj=1,2,⋯,m.

通过线性投影, 我们将若干个视觉单词嵌入一个视觉句子中.

Yi=[yi,1,yi,2,⋯ ,yi,m],yi,j=FC(Vec(xi,j))Y^{i}=[y^{i,1},y^{i,2},\cdots,y^{i,m}],\qquad y^{i,j}=FC(Vec(x^{i,j}))Yi=[yi,1,yi,2,⋯,yi,m],yi,j=FC(Vec(xi,j))

其中 yi,j∈Rcy^{i,j}\in\R^{c}yi,j∈Rc 是第 jjj 个词嵌入向量, ccc 是词嵌入维度, Vec(⋅)Vec(\cdot)Vec(⋅) 是向量操作.

TNTTNTTNT 中有两个数据流, 一个是对视觉句子的数据操作, 另一个是对每个视觉句子内的视觉单词的数据操作.

对于视觉单词, 我们使用 transformer block 计算视觉单词之间的关联:

Y′li=Yl−1i+MSA(LN(Yl−1i))Yli=Y′li+MLP(LN(Y′li))\begin{aligned} {Y^{\prime}}_{l}^{i}=&Y_{l-1}^{i}+MSA(LN(Y_{l-1}^{i}))\\ Y_{l}^{i}=&{Y^{\prime}}_{l}^{i}+MLP(LN({Y^{\prime}}_{l}^{i}))\\ \end{aligned}Y′li​=Yli​=​Yl−1i​+MSA(LN(Yl−1i​))Y′li​+MLP(LN(Y′li​))​

其中 l=1,2,⋯ ,Ll=1,2,\cdots,Ll=1,2,⋯,L 为 transformer block 的编号, LLL 为 block的总数. 第一个 block 的输入 Y0i=YiY_{0}^{i}=Y^{i}Y0i​=Yi. 经过第 lll 个 block 作用后一张图的所有词嵌入为 Yl=[Yl1,Yl2,⋯ ,Yln]\mathcal{Y}_{l}=[Y_{l}^{1},Y_{l}^{2},\cdots,Y_{l}^{n}]Yl​=[Yl1​,Yl2​,⋯,Yln​]. 这可以被视为 inner transformer block, 记为 TinT_{in}Tin​.

在视觉句子层面, 句子嵌入为 Z0=[Zclass,Z01,Z02⋯ ,Z0n]∈R(n+1)×d\mathcal{Z}_{0}=[Z_{class},Z_{0}^{1},Z_{0}^{2}\cdots,Z_{0}^{n}]\in\R^{(n+1)\times d}Z0​=[Zclass​,Z01​,Z02​⋯,Z0n​]∈R(n+1)×d, 其中 ZclassZ_{class}Zclass​ 表示类别 Token. 对于每一层, 将单词嵌入加入到句子嵌入中, 其操作为:

Zl−1i=Zl−1i+FC(Vec(Yli))Z_{l-1}^{i}=Z_{l-1}^{i}+FC(Vec(Y_{l}^{i}))Zl−1i​=Zl−1i​+FC(Vec(Yli​))

其中 Zl−1i∈RdZ_{l-1}^{i}\in\R^{d}Zl−1i​∈Rd, 全连接层 FCFCFC 使得单词嵌入和句子嵌入维度匹配. 通过加法操作, 句子嵌入的表示融合了单词嵌入的表示. 我们使用标准的 transformer block 来变换句子嵌入:

Zl′=Zl−1+MSA(LN(Zl−1))Zl=Zl′+MLP(LN(Zl′))\begin{aligned} \mathcal{Z}_{l}^{\prime}=&\mathcal{Z}_{l-1}+MSA(LN(\mathcal{Z}_{l-1}))\\ \mathcal{Z}_{l}=&\mathcal{Z}_{l}^{\prime}+MLP(LN(\mathcal{Z}_{l}^{\prime}))\\ \end{aligned}Zl′​=Zl​=​Zl−1​+MSA(LN(Zl−1​))Zl′​+MLP(LN(Zl′​))​

这可以被视为 outer transformer block, 记为 ToutT_{out}Tout​.

因此, TNT 可以形式化为:

Yl,Zl=TNT(Yl−1,Zl−1)\mathcal{Y}_{l},\mathcal{Z}_{l}=TNT(\mathcal{Y}_{l-1},\mathcal{Z}_{l-1})Yl​,Zl​=TNT(Yl−1​,Zl−1​)

在 TNT block 中, inner transformer block 用于计算同一个视觉句子内的视觉单词之间的关联, outer transformer block 用于计算视觉句子之间的关联. 通过 LLL 次堆叠 TNT block, 我们构建了 transformer-in-transformer network. 最后将类别 Token 经过全连接层用于分类.

Position encoding. 对于视觉句子增加位置编码:

Z0←Z0+Esentence\mathcal{Z}_{0}\leftarrow\mathcal{Z}_{0}+E_{sentence}Z0​←Z0​+Esentence​

其中 Esentence∈R(n+1)×dE_{sentence}\in\R^{(n+1)\times d}Esentence​∈R(n+1)×d.

对于视觉单词增加位置编码:

Y0i←Y0i+Eword,i=1,2,⋯ ,nY_{0}^{i}\leftarrow Y_{0}^{i}+E_{word},\qquad i=1,2,\cdots,nY0i​←Y0i​+Eword​,i=1,2,⋯,n

其中 Eword∈Rm×cE_{word}\in\R^{m\times c}Eword​∈Rm×c.

这样, 句子位置编码可以保持全局空间信息, 而单词位置编码用于保持局部相对位置信息.
