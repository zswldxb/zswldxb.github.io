YOLO9000: Better, Faster, Stronger

Preface

Paper

Purpose

本文提出的模型是对之前的工作 YOLO 的改进, 改进后的模型 YOLOv2 在标准检测任务上是 state-of-the-art. 此外本文还在提出的 YOLOv2 模型的基础上, 提出了 Darknet-19, YOLO9000.

Better

本节提出了 YOLOv2 模型. YOLOv2 是在 YOLO 的基础上, 对其进行一系列调整而得到的.

Batch Normalization

Batch Normalization 可显著改善收敛性, 同时消除了对其他形式正则化的需求. 批量正规化也有助于规范化模型.

High Resolution Classifier

所有 state-of-the-art 的检测方法都使用在 ImageNet 上预先训练的分类器. 与 YOLO 先在分辨率为 224×224224\times224224×224 的图像上训练带有分类器的模型, 然后将分辨率提高到 448×448448\times 448448×448, 并对模型进行调整以用于检测不同, YOLOv2 直接在分辨率为 448×448448\times 448448×448 的图像上微调带有分类器的模型 10 epochs, 然后将模型进行调整并用于检测.

Convolutional with Anchor Boxes

Faster R-CNN 中的 RPN 仅使用卷积层来预测 anchor 的偏移量和置信度, 由于预测层是卷积层, 因此 RPN 在特征图上的每个位置预测这些偏移量和置信度. 预测偏移而不是坐标可以简化问题并使网络更容易学习.

本文首先将 YOLO 的全连接层移除, 然后使用 anchor 来预测边界框. 然后减少了一层池化层, 将网络的输入数量由 448×448448\times448448×448 缩小为 416×416416\times 416416×416, 使得经过网络的输出 feature map 的大小为缩小为原来的 323232 倍后正好为 13×1313\times 1313×13. 这样物体的中心位置正好为一个像素. 此外, 本文将类预测机制与空间位置分离, 为每个 anchor 预测类别.

对于将图像分为 S×SS\times SS×S 个网格, 每个网格预测 BBB 个 box(5 个参数), CCC 个类别的 YOLO 模型的输出大小为 S×S×(5×B+C)S\times S\times(5\times B+C)S×S×(5×B+C).

而对于将图像输出长宽为 S×SS\times SS×S , 每个输出位置预测 BBB 个 anchor(5 个参数), 一共 CCC 个类别的 YOLOv2, 其输出大小为 S×S×(B×(5+C))S\times S\times (B\times (5+C))S×S×(B×(5+C)).

将 YOLO 和 anchor 结合时存在两个问题, 将在 Dimension Clusters 和 Direct Location Prediction 中解决.

Dimension Clusters

第一个问题是 box 的大小是 hand picked. 网络是可以适当调整框, 但是如果为网络挑选更好的先验, 则网络可以更容易学习到较好的表示.

这里采用在训练集 bounding box 上运行 k-means 聚类来找到较好的先验, 然而经典的 k-means 使用的距离度量为欧氏距离, 较大的 box 相较于较小的 box 可能会产生更多错误, 这与我们希望的具有良好的 IOU 分数的先验(与框的大小无关)不同. 因此定义距离度量为

d(box,centroid)=1−IOU(box,centroid)d(box,centroid)=1-IOU(box, centroid)d(box,centroid)=1−IOU(box,centroid)

本文经过实验, 选定 k=5k=5k=5.

Fine-Grained Features

Faster R-CNN 和 SSD 在网络中的各种特征上运行其提议网络, 以获得一系列分辨率. 本文借鉴这一思想, 将早期的 26×2626\times 2626×26 的分辨率的特征通过 passthrough layer 与 13×1313\times 1313×13 的特征图在通道维度上进行拼接, 如将 26×26×51226\times 26\times 51226×26×512 的 feature map 变为 13×13×204813\times 13\times 204813×13×2048 的特征图, 然后与 13×13×c13\times 13\times c13×13×c 的特征图在通道维度上进行拼接.

Multi-Scale Training

由于模型只是用了卷积层和池化层, 因此可以动态调整大小. 我们希望 YOLOv2 可以在处理不同尺度的图像时具有鲁棒性. 本文没有固定输入图像的大小, 而是每隔几次迭代就改变网络的输入图像大小. 由于本文模型下采样了 323232 倍, 因此, 本文采用的输入图像大小的为 {(320+i×32)2}i=09\{(320+i\times 32)^{2}\}_{i=0}^{9}{(320+i×32)2}i=09​, 最小的大小为 320×320320\times 320320×320, 最大的大小为 608×608608\times 608608×608.

这样的机制使得网络在不同输入大小的图像上都具有良好的预测(同一个网络可以预测不同分辨率下的检测).

Faster

本文希望检测准确，也希望检测速度快。因此提出了 Darknet-19.

Darknet-19

本文提出了一个全新的分类模型, 称为 Darknet-19, 将全部全连接层替换为全局平均池化层. 使用 1×11\times 11×1 卷积来压缩 3×33\times 33×3 卷积之间的特征表示. 私用批量归一化来稳定训练, 加速收敛, 并对模型进行正则化. 网络架构如表所示.

Training for classification

Darknet-19 首先在标准 ImageNet 上训练(输入大小为 224×224224\times224224×224), 在训练期间使用数据增强, 包括随机裁剪、旋转、色调、饱和度和曝光偏移. 然后在输入大小为 448×448448\times 448448×448 的网络上进行微调.

Training for detection

我们删除 Darknet-19 的最后一个卷积层来修改网络以进行检测, 然后添加 333 个 3×33\times 33×3 的卷积层, 每个卷积层的通道数为 102410241024, 最后是一个 1×11\times11×1 卷积, 其通道数为 B×(5+C)B\times(5+C)B×(5+C). 此外还在最后的 3×3×5123\times 3\times5123×3×512 到倒数第二个卷积层添加一个直通层, 以便模型使用细粒度特征.

Stronger

本文提出了一种联合训练分类和检测数据的机制. 本文的方法使用标记为检测的图像来学习特定于检测的信息, 例如边界框坐标预测和对象性以及如何对常见对象进行分类. 使用只有类别标签的图像来扩展其可以检测的类别数量. 在训练期间, 我们混合来自检测和分类数据集的图像. 当网络看到标记为检测的图像时, 可以基于完整的 YOLOv2 损失函数进行反向传播损失. 当它看到分类图像时, 网络只从其特定分类部分进行反向传播损失.

本文采用分类数据集 ImageNet 和 检测数据集 COCO,由于 ImageNet 与 COCO 的类别有重合, 而使用 softmax 时假设这些类别互斥, 这给组合数据集带来了一些问题, 为了实现联合训练, 本文使用 multi-label model 来组合不互斥的的数据集.

Hierarchical classification

由于 ImageNet 的标签是从 WordNet 中提取的, 这些类别之间的关系为有向图, 而非树, 这里通过将 ImageNet 中的概念构建为层次树来简化问题. 构建后的概念称为 WordTree.

为了使用 WordTree 进行分类, 我们预测每个节点的条件概率, 以获得给定同义词集的"同义词集的每个下位词"的概率. 如对于同义词集为"terrier", 我们计算:

Pr(Norfolk terrier|terrier)Pr(Yorkshire terrier|terrier)Pr(Bedlington terrier|terrier)\begin{aligned} &\text{Pr(Norfolk terrier|terrier)}\\ &\text{Pr(Yorkshire terrier|terrier)}\\ &\text{Pr(Bedlington terrier|terrier)}\\ \end{aligned}​Pr(Norfolk terrier|terrier)Pr(Yorkshire terrier|terrier)Pr(Bedlington terrier|terrier)​

而当要计算一个样本为某个类别的概率时, 使用条件概率公式得到其概率. 如要计算一个样本是"Norfolk terrier" 的概率, 我们计算:

Pr(Norfolk terrier)=Pr(Norfolk terrier|terrier)×Pr(terrier|hunting dog)⋯×Pr(mammal|Pr(animal)×Pr(animal|physical object)\begin{aligned} \text{Pr(Norfolk terrier)}&=\text{Pr(Norfolk terrier|terrier)}\\ &\times \text{Pr(terrier|hunting dog)}\\ &\cdots\\ &\times \text{Pr(mammal|Pr(animal)}\\ &\times \text{Pr(animal|physical object)}\\ \end{aligned}Pr(Norfolk terrier)​=Pr(Norfolk terrier|terrier)×Pr(terrier|hunting dog)⋯×Pr(mammal|Pr(animal)×Pr(animal|physical object)​

这里假设 Pr(physical object)=1\text{Pr(physical object)}=1Pr(physical object)=1

此外, 此时不再假设每个图像都有一个目标, 而是使用 YOLOv2 的目标检测器来提供 Pr(physical object)\text{Pr(physical object)}Pr(physical object). 检测器预测边界框和概率树. 我们向下遍历树, 在每次拆分时采用最高置信度路径, 直到达到某个阈值并预测该对象类别.

Joint classification and detection

此外本文使用完整的 ImageNet 的前 900090009000 类组成的数据集和 COCO 检测数据集相结合. 构建了 YOLO9000, 同时对 COCO 数据集采用过采样的方法来平衡 ImageNet 和 COCO 的样本数量之比, 使其比例为 4:14:14:1.

YOLO9000 提出的目的是为了使得模型可以同时进行分类和检测.
