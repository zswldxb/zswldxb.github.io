End-to-End Object Detection with Transformers

Preamble

论文: End-to-End Object Detection with Transformers

Motivation

现有的绝大多数目标检测算法不是端到端的模型, 都或多或少使用手工设计的组件(如 non-maximum suppression, anchor generation), 这导致这些算法的设计较为复杂且其性能受这些预处理或后处理组件的影响较大. 为了解决这一问题, 本文提出了端到端的目标.

Method

Object detection set prediction loss

对于任意一张给定的图像, DETR 假设其中最多存在 NNN 个 objects. 因此其通过模型预测 NNN 个 objects y^\hat{y}y^​, 并与 ground truth(GT) 进行最佳匹配, 并计算损失. 由于 GT 中的 object 数量可能小于 NNN 个. 因此对 GT 增加一些 ∅\varnothing∅ (no object) 使得 GT 中的 object 总数为 NNN 得到 yyy. 并通过二部图匹配算法 (匈牙利匹配) 计算 yyy 与 y^\hat{y}y^​ 的损失.
