Norms, including the LpL^{p}Lp norm, are functions mapping vectors to non-negative values. On an intuitive level, the norm of a vector xxx measures the distance from the origin to the point xxx. More rigorously, a norm is any function fff that satisﬁes the following properties:

范数（包括 LpL^{p}Lp 范数）是将向量映射到非负值的函数。直观上来说，向量 xxx 的范数衡量从原点到点 xxx 的距离。更严格地说，范数是满足下列性质的任意函数：

  • f(x)=0⇒x=0f (x) = 0 \Rightarrow x = 0f(x)=0⇒x=0

The L2L^{2}L2 norm, with p=2p= 2p=2, is known as the Euclidean norm, which is simply the Euclidean distance from the origin to the point identiﬁed by xxx. The L2L^{2}L2 norm is used so frequently in machine learning that it is often denoted simply as ∥x∥\parallel x\parallel∥x∥, with the subscript 2 omitted. It is also common to measure the size of a vector using the squared L2L^{2}L2 norm, which can be calculated simply as xTxx^{T}xxTx.

The squared L2L^{2}L2 norm is more convenient to work with mathematically and computationally than the L2L^{2}L2 norm itself. For example, each derivative of the squared L2L^{2}L2 norm with respect to each element of xxx depends only on the corresponding element of xxx, while all the derivatives of the L2L^{2}L2 norm depend on the entire vector. In many contexts, the squared L2L^{2}L2 norm may be undesirable because it increases very slowly near the origin. In several machine learning applications, it is important to discriminate between elements that are exactly zero and elements that are small but nonzero. We sometimes measure the size of the vector by counting its number of nonzero elements. Some authors refer to this function as the “L0L^{0}L0 norm,” but this is incorrect terminology. The number of nonzero entries in a vector is not a norm, because scaling the vector by α\alphaα does not change the number of nonzero entries. The L1L^{1}L1 norm is often used as a substitute for the number of nonzero entries.

有时候我们会统计向量中非零元素的个数来衡量向量的大小。有些作者将这种函数称为“L0L^{0}L0 范数”，但是这个术语在数学意义上是不对的。向量的非零元素的数目不是范数，因为对向量缩放 α\alphaα 倍不会改变该向量非零元素的数目。因此，L1L^{1}L1范数经常作为表示非零元素数目的替代函数。
