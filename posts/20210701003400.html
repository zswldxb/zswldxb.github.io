Attention Is All You Need

Preface

Paper

Motivation

循环神经网络中的顺序性导致了模型计算缓慢, 本文提出了一个全新的网络架构 Transformer, 以取消模型执行时的顺序性.

Embedding

由于 Inputs 中的每个元素都为输入字典中的某个 Token 的编号, 而编号的表示能力是很差的, 因此需要通过 Input Embedding\text{Input Embedding}Input Embedding 将每个编号映射为一个词向量. 同理输出 Outputs 中的每个元素都为输出字典中某个 Token 的编号, 通过 Output Embedding\text{Output Embedding}Output Embedding 将每个编号映射为一个词向量.

由于 Transformer 中使用并行执行取代顺序执行, 因此需要额外考虑一个序列中的每个 Token 之间的位置关系. 通过对每个词向量增加位置编码信息以表示同一个序列中不同的 Token 之间的位置关系.

Positional Encoding

  1. 希望具有平移不变性, 即一个序列中的第 pospospos 个 Token 与第 pos+kpos+kpos+k 个 Token 之间的距离仅依赖于 kkk, 而与具体位置无关.
