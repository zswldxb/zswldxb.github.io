Excel_列标字母转数字

  1. 打开Excel，我们可以看一下，行号是数字，列标是字母。如果希望全部都变成数字的话，就需要将列标的字母变成数字。
  2. 点击"文件"。
  3. 点击"选项"。
  4. 点击"公式"。
  5. 勾选"R1C1引用样式"，点击确定。即可实现列标字母转数字。

Python3_and和or运算

Python3_and和or运算

在 Python 中, and 和 or 执行布尔逻辑运算, 但是其不返回布尔值, 而是返回参与运算的数.

在 Python 中, 0, '', [], (), {}, None 为假, 其余都为真. (可以在类中定义特定的方法使得类实例为假）

and 运算: and 运算返回第一个为假的元素, 若参与运算的元素都为真, 则返回最后一个元素.

or 运算: or 运算返回第一个为真的元素, 若参与运算的元素都为假, 则返回最后一个元素.

在 Python 中 and 与 or 运算优先级相同(C++ 中 && 优先级高于 ||), 一律从左向右执行, 但是采取短路执行方式.
