Pytorch.torch.quantize_per_tensor

quantize_per_tensor 返回量化张量。

量化（Quantization）就是将机器中的使用浮点 tensor 转化为整数 tensor 使得计算变得更快，并节省资源消耗，而且这种方法不会使准确度下降很多。

input 表示要量化的 tensor。

scale 表示张量公式中的缩放大小。

zero_point 表示偏移量，该值映射为浮点数零。

dtype 表示返回的张量数据类型，必须为量化张量类型。

量化公式：

output = round(\dfrac{input}{\text{scale}}+\text{zero_point})

Pytorch.torch.as_strided

as_strided 根据现有 tensor 和给定步长创建一个视图。视图与原数据共享内存。因此，改变一个视图将改变其他所有视图以及原数据，改变原数据，视图也将相应改变。

Parameters

  • input 表示输入 tensor。

  • size 表示生成的视图的大小。

  • stride 表示输出张量的步长。

  • storage_offset 表示输出张量在原张量中的偏移量。
