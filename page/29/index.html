Preface

Paper

Purpose

本文指出: 本文思路与此前的目标检测的思路不同, 本文思路将目标检测当作回归问题处理, 直接根据网络预测边界框和类概率. 本文提出的网络名取为 YOLO.

YOLO 相比于其他目标检测方法的优点:

  1. YOLO 的检测速度很快.
  2. YOLO 在进行预测时是基于整个图像的.
  3. YOLO 学习到了目标的通用表示.
  4. YOLO 是 end-to-end model.

YOLO 相比于其他目标检测方法的缺点:

  1. 对边界框预测强加了空间约束, 因为每个网格单元只能预测两个框, 并且只能有一类. 这大大限制了模型可以预测的目标数量.
  2. 模型定位不准确.

Mask TextSpotter: An End-to-End Trainable Neural Network for Spotting Text with Arbitrary Shapes

Preface

Paper

Purpose

本文指出:

  1. 此前的大多数工作, 都是将文本检测和识别分开处理的. 然而这两个任务高度相关和互补. 一方面, 检测质量的好坏决定了识别的准确性；另一方面, 识别结果可以提供反馈, 提高检测质量.
  2. 考虑两个任务一起处理的工作, 它们不能完全以端到端的方式进行训练, 主要有两个原因:
    1. 一个是文本识别部分需要准确的位置进行训练, 而早期迭代中的位置通常不准确.
    2. 另一个是采用的 LSTM 或 CTC loss 比一般的 CNN 难优化. 而采用这些方式的文章大多是处理水平或定向文本, 还没有考虑真实场景中弯曲形状的文字.

本文提出的模型用于实现端到端(同时实现文本检测和识别)的对于任意形状的文本实例的检测与识别.

Function

  • axes: (2,) array_like
    • The array is rotated in the plane defined by the axes. Axes must be different.

Returns
